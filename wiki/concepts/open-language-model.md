## Definition

**Open Language Model** (开放语言模型) — a language model release that exposes enough artifacts, such as weights, data, code, and logs, to support independent inspection, reproduction, and scientific study.

## Key Points

- OLMo defines openness as releasing the whole framework, not just final checkpoints or inference code.
- The release includes pretrained weights, `500+` intermediate checkpoints, training logs, Dolma data artifacts, evaluation code, and adaptation code.
- The paper argues that scientific work on bias, risk, optimization, and data effects is blocked when model creators hide training details.
- OLMo releases its code and weights under Apache 2.0 to allow both research and commercial use.
