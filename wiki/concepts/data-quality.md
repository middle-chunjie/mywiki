## Definition

**Data Quality** (数据质量) — the degree to which a dataset is accurate, relevant, and aligned with the target task and evaluation criteria.

## Key Points

- [[cui-2022-codeexp-2211-15395]] shows that better-filtered data can outperform much larger but noisier raw corpora for code explanation generation.
- The refined partition contains only about one fifteenth as many examples as the raw partition, yet it yields better human and automatic scores.
- The paper attributes raw-data weakness to the prevalence of short one-line docstrings that behave more like summaries than explanations.
- Human annotation criteria for adequacy, coverage, and coherence are used as the quality target for dataset refinement.
