## Definition

**Agentic Data Synthesis** (智能体式数据合成) — the generation of benchmark or training data by delegating planning, retrieval, and validation steps to tool-using agents instead of hand-written templates alone.

## Key Points

- InfoMosaic-Flow is an agentic synthesis pipeline that uses planning, tool calls, and evidence integration to create tasks.
- Seed scenarios are grounded with tool outputs rather than uncontrolled web text, improving traceability.
- Human review is layered on top of the automated pipeline to improve factual alignment, coherence, and difficulty.
