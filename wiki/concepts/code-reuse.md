## Definition

**Code Reuse** (代码复用) — the practice of reapplying previously written code fragments or abstractions in later implementations instead of rewriting them from scratch.

## Key Points

- CodeChain treats cluster-centroid sub-modules from previous samples as reusable building blocks for later generations.
- The framework is explicitly inspired by how human developers revise programs by adapting code from earlier attempts.
- Reuse is applied at the sub-module level, not the full-program level, because fully correct whole programs are rare on hard benchmarks.
- The paper links improved correctness to improved reuse because representative modules preserve useful partial solutions across rounds.
