## Definition

**Mixed-Initiative Interaction** (混合主动交互) — an interaction pattern in which both the user and the system can take initiative in advancing a search or dialogue.

## Key Points

- The paper positions proactive retrieval as a mixed-initiative alternative to the dominant query-response paradigm.
- It contrasts proactive interventions with clarifying questions, another well-studied mixed-initiative mechanism in CIS.
- A proactive system must infer when system initiative adds value instead of waiting for explicit user action.
