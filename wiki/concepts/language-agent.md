## Definition

**Language agent** — an agent whose planning, communication, and environment-facing actions are mediated primarily through a language model that interprets natural-language goals and emits executable action sequences.

## Key Points

- The paper treats language agents as systems that must combine human interaction, reasoning, planning, and environment grounding.
- It argues that strong natural-language ability alone is insufficient because agents must also produce valid code or API-like actions.
- Lemur evaluates language-agent capability through tool use, environment-feedback self-debugging, natural-language feedback following, and partially observable exploration.
- The paper shows that balancing text and code ability improves agent performance more than specializing in only one of the two domains.
