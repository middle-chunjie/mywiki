## Definition

**Programming Language Processing** (编程语言处理) — the study of computational methods for understanding, representing, and analyzing source code using techniques from machine learning, program analysis, and natural language processing.

## Key Points

- [[long-2022-multiview]] treats program representation as a central problem inside PLP because many downstream tasks depend on code embeddings.
- The paper lists code retrieval, code annotation, bug prediction and fixing, and program translation as representative PLP applications.
- It argues that PLP models should capture program semantics such as data flow and control flow rather than rely mainly on sequential tokens or syntax trees.
- Algorithm detection is used in the paper as a challenging PLP benchmark because programs can contain multiple interacting algorithms and data structures.
