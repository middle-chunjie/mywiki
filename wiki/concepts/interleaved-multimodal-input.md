## Definition

**Interleaved Multimodal Input** (交错多模态输入) — an input format where text, images, sub-images, and other multimodal elements are mixed in sequence so the model must integrate them jointly.

## Key Points

- GPT-4V is described as handling arbitrarily interleaved images and text rather than only fixed single-image prompts.
- The report uses this format for tasks such as combining menu information with food images, multi-image reasoning, and richer instruction delivery.
- The same capability underpins multimodal demonstration-based prompting because examples and queries can be mixed in one prompt stream.
- The authors argue this flexibility is a major differentiator from earlier vision-language models that were constrained by image count or rigid formatting rules.
