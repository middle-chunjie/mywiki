## Definition

**Model Context Protocol** (模型上下文协议) — a standardized interface for exposing external tools and data sources to language-model agents in a structured, callable form.

## Key Points

- The paper treats MCP as the infrastructure shift that expands agents beyond web-only retrieval into heterogeneous domain tool ecosystems.
- InfoMosaic-Bench includes `77` MCP tools spanning domains such as medicine, finance, maps, and video.
- The benchmark tests not only whether tools exist, but whether agents can select and integrate MCP tools reliably.
