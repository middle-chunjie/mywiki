## Definition

**Domain-Specific Tool** (领域专用工具) — an external tool specialized for a particular domain, exposing structured data or operations that generic web search cannot reliably provide.

## Key Points

- The benchmark uses domain tools for medicine, finance, maps, video, and other settings to test access to non-web-native information.
- Domain tools provide selective gains, improving some domains while hurting others when agents misuse or mis-select them.
- A central claim of the paper is that tool availability is not enough; tool planning and invocation competence are the actual bottlenecks.
