## Definition

**Tool Use** (工具使用) — the capability of an intelligent system to invoke external software components and integrate their outputs into a larger reasoning or execution process.

## Key Points

- The paper positions tool use as the mechanism that turns LLMs from pure text predictors into agents able to carry out complex multi-step workflows.
- It argues that many scientific tasks are blocked less by reasoning than by the absence of already integrated specialized tools.
- TOOLMAKER extends the tool-use paradigm by generating new executable tools from repository-backed research artifacts rather than only selecting from a fixed catalog.
- The benchmark tasks require practical tool use across installation, configuration, execution, and debugging rather than simple API argument filling.
