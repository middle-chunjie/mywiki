## Definition

**Data Types** (数据类型) — the set of type constraints that determine which values and operations are valid for program variables and expressions.

## Key Points

- [[hooda-2024-do-2402-05980]] frames data types as one of the core Programming Concept Predicates relevant to code generation.
- The paper motivates type-related understanding as necessary for generating code that respects the underlying language's type system.
- Although data types are part of the conceptual taxonomy, the paper's instantiated mutations focus more heavily on naming, control flow, and data flow than on rich type-system transformations.
