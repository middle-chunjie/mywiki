## Definition

**Generative Recommender System** (生成式推荐系统) — a recommender system that directly generates candidate items or text for a user context instead of selecting only from a fixed item catalog.

## Key Points

- This paper studies recommendation as open-set question generation, where the output space is effectively unconstrained.
- The proposed system improves generated recommendations through iterative prompting rather than model retraining.
- User engagement is represented by question-level CTR, which is used to decide which generated items to keep or replace.
- The paper argues that generative recommenders can better adapt to evolving domains than rankers tied to static candidate pools.
