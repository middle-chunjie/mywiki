## Definition

**Software Engineering Agent** (软件工程智能体) — an agent that interacts with a repository, tools, and execution feedback to resolve real software tasks such as GitHub issues by producing code changes.

## Key Points

- The paper treats SWE agents as systems that map from an issue description plus repository state to a patch validated by tests.
- It studies both a general-purpose scaffold ([[openhands]]) and a specialized workflow scaffold ([[moatlesstools]]).
- SWE-Gym is proposed specifically to train such agents rather than only evaluate them.
- Fine-tuning on successful SWE-Gym trajectories substantially improves open-weight agent resolution rate on SWE-Bench.
