## Definition

**Agent-Environment Interaction** (智能体-环境交互) — the exchange between an agent and an external environment in which the agent acts and the environment returns observable consequences.

## Key Points

- [[wang-2024-executable-2402-01030]] distinguishes agent-environment interaction from agent-user dialogue in a three-role framework of user, agent, and environment.
- CodeAct is proposed specifically for this channel: actions are executable Python snippets and observations are execution results or errors.
- CodeActInstruct focuses on this interaction mode across information seeking, software use, external memory, and robot planning.
- The paper argues that training on this interaction pattern improves agent capability without sacrificing general conversational ability when mixed with dialogue data.
