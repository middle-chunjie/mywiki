## Definition

**Query Breakdown** (查询拆解) — the process of rewriting a complex user request into multiple narrower subqueries that are easier for retrieval systems or search engines to satisfy.

## Key Points

- The Web-Search agent uses query breakdown before hitting the search engine.
- The decomposition step is conditioned not only on the raw search request but also on the broader reasoning context from Mind-Map.
- The paper shows that adding query breakdown consistently improves search-based reasoning performance.
- Query breakdown is most effective when combined with reranking and Mind-Map context rather than used as an isolated retrieval heuristic.
