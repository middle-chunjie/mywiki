## Definition

**Code Intelligence** (代码智能) — the use of learned or algorithmic program representations to support automated understanding, retrieval, analysis, and generation of code.

## Key Points

- The paper presents code representation learning as a foundational step toward broader code-intelligence tasks.
- FuzzPretrain is validated on multiple code-intelligence benchmarks, including code search, clone detection, defect detection, and text-to-code search.
- Its central claim is that combining structural and execution information produces more discriminative embeddings for downstream code understanding.
- The work does not show gains on code generation, indicating that not all code-intelligence tasks benefit equally from execution-aware pre-training.
