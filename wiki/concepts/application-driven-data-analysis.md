## Definition

**Application-Driven Data Analysis** (应用驱动数据分析) — data analysis that answers a stakeholder's decision-oriented query by grounding multi-step quantitative reasoning in the available database and synthesizing actionable findings.

## Key Points

- DACO frames each task around a stakeholder role and a concrete decision or investigation goal rather than a simple factual lookup.
- Outputs are expected to contain both findings and suggestions, so the target is analytic usefulness instead of only answer correctness.
- Solving the task requires decomposing a query into multiple perspectives, executing computations over tables, and interpreting the results in natural language.
- The paper positions this setting as substantially harder than standard table QA benchmarks focused on retrieval, aggregation, or short-form reasoning.
