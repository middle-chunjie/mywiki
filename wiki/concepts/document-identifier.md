## Definition

**Document Identifier** (文档标识符) — a symbolic token or string that uniquely denotes a document so a retrieval system can map a query-time prediction back to the corresponding item.

## Key Points

- The paper uses free-text docids rather than numeric identifiers or URLs as the retrieval target generated by the model.
- During indexing, docids are generated from pseudo queries rather than copied from raw document text.
- The docid bank is de-duplicated so each valid docid maps to a unique document at retrieval time.
- Retrieval is constrained so the LLM can only emit identifiers that exist in the bank, reducing invalid outputs.
