## Definition

**Multilingual Benchmark** (多语言基准) — a benchmark that evaluates the same or aligned capabilities across multiple languages under a shared data and scoring protocol.

## Key Points

- [[unknown-nd-xcodeevalan-2303-03004]] frames xCODEEval as a multilingual benchmark for code understanding, generation, translation, repair, and retrieval.
- The benchmark supports `11` executable languages in its main tasks and extends retrieval analysis to `17` languages.
- Shared problem structure lets the paper compare monolingual and cross-lingual behavior under the same executable evaluation setup.
- Language-resource imbalance is itself a measured property of the benchmark, with lower-resource languages showing worse retrieval and synthesis performance.
- [[exec-eval]] provides the shared runtime layer that makes cross-language executable comparison possible.
