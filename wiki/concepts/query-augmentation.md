## Definition

**Query Augmentation** (查询增强) — the process of enriching an original query with additional text intended to improve downstream retrieval or ranking quality.

## Key Points

- This paper augments each query with multiple answer-like passages generated by an LLM rather than with isolated keywords or embeddings.
- The augmented query is formed as `Concat(q, a_1^q, q, a_2^q, ..., q, a_N^q)`, so the original query is repeated alongside each generated answer.
- The authors argue that augmentation works better when the LLM is conditioned on retrieved in-domain candidates instead of only the short user query.
- Because the final retriever is BM25, the augmented text is consumed literally through lexical overlap rather than through a learned dense encoder.
