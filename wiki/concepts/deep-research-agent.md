## Definition

**Deep-Research Agent** (深度研究智能体) — an autonomous agent that performs multi-step information seeking, tool use, synthesis, and answer generation over open external sources.

## Key Points

- IterResearch is positioned as a new paradigm for deep-research agents rather than a single prompting trick.
- The agent interacts with tools such as search, browser, scholar, and Python through structured action outputs.
- The paper argues that deep-research agents need memory management and selective forgetting, not just larger context windows.
- Benchmarks are designed to test web navigation, evidence synthesis, and long-horizon question answering rather than closed-book inference only.
