## Definition

**Ad-hoc Retrieval** (即席检索) — the retrieval setting where a system ranks a collection for an incoming query without a persistent task-specific interaction history.

## Key Points

- The paper situates DL-MIA within ad-hoc web search evaluation, where standard benchmarks often expose only short keyword queries plus relevance labels.
- TREC-DL query sets are used as the base ad-hoc retrieval benchmark from which intent-aware annotations are derived.
- The authors argue that conventional ad-hoc evaluation misses whether a ranker actually captures the user's intended aspect of an ambiguous query.
- DL-MIA extends ad-hoc retrieval evaluation with intent-conditioned labels rather than replacing the underlying retrieval task.
