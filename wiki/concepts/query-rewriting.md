## Definition

**Query Rewriting** (查询改写) — transforming an original query into a clearer or more task-effective formulation while preserving or sharpening the intended information need.

## Key Points

- The paper frames explicit user intents as reformulations that expose aspects hidden in the original short query.
- DL-MIA is proposed as a useful evaluation set for query rewriting systems, especially when the goal is user-machine intent alignment.
- Intent descriptions are deliberately descriptive and constrained to short phrases rather than direct answers, making them suitable as rewrite targets.
- The experiments show that using explicit intents as queries yields better ranking quality than using the original ambiguous queries.
