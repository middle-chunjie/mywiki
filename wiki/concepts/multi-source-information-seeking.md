## Definition

**Multi-Source Information Seeking** (多源信息寻求) — the process of answering a query by retrieving and integrating evidence from multiple heterogeneous information sources rather than a single search channel.

## Key Points

- InfoMosaic-Bench is explicitly designed so tasks require combining general web evidence with domain-specific tools.
- The benchmark rejects shortcut cases where one clue or one source is sufficient to recover the answer.
- Each accepted item includes condition-level labels and tool traces so multi-source behavior can be analyzed diagnostically.
