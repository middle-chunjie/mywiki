## Definition

**Code Debugging** (代码调试) — the task of identifying and correcting faults in source code so the program better satisfies its intended behavior.

## Key Points

- [[luo-2023-wizardcoder-2306-08568]] adds code debugging as one of the new code-specific instruction-evolution directions when adapting [[evol-instruct]].
- The paper treats debugging-oriented instructions as a distinct capability missing from generic instruction-tuning recipes.
- By injecting debugging-style tasks into synthetic supervision, WizardCoder broadens beyond plain direct code synthesis toward error-aware code assistance.
