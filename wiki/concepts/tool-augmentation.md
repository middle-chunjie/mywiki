## Definition

**Tool Augmentation** (工具增强) — augmenting a language model with external tools such as search engines, knowledge graphs, APIs, or specialized models that can be invoked during generation to improve factuality or task completion.

## Key Points

- The survey treats tool augmentation as a major external-knowledge strategy for reliable response generation.
- Covered tool classes include search-engine access, knowledge-graph reasoning, API invocation, and model-based tool orchestration.
- Representative examples in the survey include ReAct, WebGPT, StructGPT, Toolformer, ToolLLM, and HuggingGPT.
- Tool augmentation expands GenIR beyond passive retrieval by enabling action, computation, and structured data access.
- A central tradeoff is whether the extra reliability or capability from tools justifies additional training and inference cost.
