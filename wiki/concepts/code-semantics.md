## Definition

**Code Semantics** (代码语义) — the meaning-bearing aspects of source code that determine how values propagate, how control executes, and what computations a program performs beyond surface syntax.

## Key Points

- [[long-2022-multiview]] is explicitly motivated by the claim that prior methods overemphasize syntax and underweight semantics.
- The paper operationalizes code semantics through data flow, control flow, and read-write interactions.
- It treats semantic separation as beneficial because mixed single-view graphs can entangle heterogeneous signals.
- The strongest empirical gains appear on the harder multi-label benchmark, supporting the idea that semantic structure matters when shallow cues are insufficient.
