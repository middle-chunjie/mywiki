## Definition

**Machine Instruction** (机器指令) — a low-level executable operation, usually consisting of an opcode plus operands, that specifies a concrete computation step in a runtime or hardware execution model.

## Key Points

- TranCS translates Java code at the machine-instruction level rather than from source tokens, AST nodes, or CFG nodes.
- The paper treats JVM bytecode instructions as a more semantically faithful basis for capturing execution behavior.
- Each instruction is paired with manually curated translation rules derived from the instruction specification.
- Operands expose constants, local-variable indices, and jump targets that help recover semantic context.
- Instruction sequences are generated by compiling and disassembling Java code before translation.
