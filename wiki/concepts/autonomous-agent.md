## Definition

**Autonomous Agent** (自主智能体) — an agentic system that uses model-based reasoning, memory, tools, or planning to pursue tasks with limited step-by-step human intervention.

## Key Points

- The paper treats modern LLM-based autonomous agents as the starting point from which stronger multi-agent systems can be automatically generated.
- It highlights planning, tool use, memory, personas, and role settings as skills that shape autonomous-agent behavior and can be modified by evolution.
- EvoAgent is evaluated not only on static NLP tasks but also on interactive environments, reflecting the paper's view that autonomy matters in open-world settings.
- The proposed pipeline keeps the initial agent as a reusable seed rather than discarding it, so autonomous behavior is progressively extended instead of rebuilt from scratch.
