## Definition

**Code Understanding** (代码理解) — the task of modeling the semantics, behavior, or intent of software artifacts from code and related information sources such as documentation and developer discussions.

## Key Points

- This paper argues that code understanding should include textual artifacts about code, not only program syntax or flow.
- BLANCA operationalizes code understanding through five downstream tasks over posts, documentation, and class relations.
- The benchmark treats improved embedding quality on these tasks as evidence of better code-understanding representations.
- Results show that fine-tuning on code-related text tasks materially improves model performance over off-the-shelf embeddings.
