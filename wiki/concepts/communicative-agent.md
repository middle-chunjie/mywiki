## Definition

**Communicative Agent** (交互式智能体) — an agent that uses natural-language interaction with humans or other agents as its main channel for coordination and task execution.

## Key Points

- CAMEL studies communicative agents in a cooperative assistant-user setting rather than as isolated single-turn models.
- Each agent is instantiated from a large language model plus a role-specific system prompt.
- The paper uses communicative agents both to solve tasks and to reveal failure modes such as role flipping and infinite loops.
- The framework is intended as a scalable research instrument for analyzing emergent behaviors in agent societies.
