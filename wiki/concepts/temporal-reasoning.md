## Definition

**Temporal Reasoning** (时间推理) — the ability to infer answers from the ordering, timing, or duration of events across one or more interaction histories.

## Key Points

- LOCOMO includes a dedicated temporal category, making temporal reasoning a core evaluation target in the paper.
- `Mem0^g` improves temporal reasoning by storing timestamped entities and relations rather than only natural-language memory snippets.
- The paper reports the best temporal J score for `Mem0^g`, indicating that structured relational memory helps event ordering and chronology.
- The prompting strategy for answering time questions explicitly converts relative references into specific dates or periods.
