## Definition

**Source Code Authorship Attribution** (源代码作者归属) — the task of inferring the author of a program from stylistic, structural, or semantic cues present in source code.

## Key Points

- The paper studies single-authorship attribution, where each program is assigned to one author from a finite candidate set.
- RoPGen treats attribution robustness as a central requirement because high clean accuracy alone is insufficient under style manipulation or adversarial attack.
- The evaluated attribution systems are DL-CAIS and PbNN, covering token-based and path-based program representations.
- The paper frames source code authorship attribution as useful for software forensics, plagiarism analysis, bug fixing, and software quality analysis.
