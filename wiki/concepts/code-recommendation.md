## Definition

**Code Recommendation** (代码推荐) — the task of matching developers with source-code files or development tasks by combining behavioral, structural, and semantic signals from software repositories.

## Key Points

- [[jin-2023-code]] formulates code recommendation as predicting future user-file contribution behavior in open-source repositories.
- The paper argues that this task differs from standard recommendation because files have code semantics and live inside project hierarchies.
- CODER jointly models microscopic user-file interactions and macroscopic user-project behaviors instead of relying on a single interaction matrix.
- The benchmark includes intra-project, cross-project, and cold-start settings to reflect realistic OSS contribution scenarios.
