## Definition

**Multimodal Recommendation** (多模态推荐) — recommendation that combines heterogeneous signal types such as text, structure, metadata, and interaction behavior to improve matching quality.

## Key Points

- [[jin-2023-code]] treats source code semantics, directory structure, and user behavior as complementary modalities for recommendation.
- The model fuses code segments with contributor-aware attention before graph propagation over the repository hierarchy.
- Project-level stars or similar repository interactions provide a second behavioral modality beyond direct code contributions.
- The paper shows that combining modalities is especially beneficial on sparse data and for cold-start users.
