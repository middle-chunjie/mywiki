## Definition

**Algorithm Detection** (算法检测) — the task of identifying which algorithms and data structures are implemented in a given piece of source code.

## Key Points

- [[long-2022-multiview]] frames algorithm detection as a core PLP task and a stringent benchmark for program representation quality.
- The paper emphasizes that one program may contain multiple labels and that one algorithm may have many implementation variants.
- It also notes that distinct algorithms can share very similar code patterns, making the task semantically demanding.
- The paper evaluates algorithm detection with both single-label problem classification on [[poj-104]] and multi-label algorithm classification on [[alg-109]].
