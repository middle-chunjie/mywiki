## Definition

**Repository-Level Coding Task** (仓库级编码任务) — a coding task defined over a full software repository, where solving the task requires understanding repository context, editing code, and validating behavior at the project level.

## Key Points

- The paper distinguishes repository-level tasks from isolated code-generation problems such as HumanEval or APPS.
- SWE-Gym tasks are derived from real GitHub issues and evaluated through repository test suites.
- The benchmark statistics show large average codebases and multi-file gold patches, underscoring realistic task complexity.
- The authors argue that these tasks better capture real-world software engineering than synthetic standalone exercises.
