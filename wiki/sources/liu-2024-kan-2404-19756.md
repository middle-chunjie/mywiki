## Summary

KAN replaces each scalar weight in an MLP with a learnable one-dimensional function, motivated by Kolmogorov-Arnold representations and implemented with spline-parameterized edge activations. The paper generalizes the original depth-2 theorem into deep architectures, introduces grid extension so spline resolution can be increased without retraining from scratch, and adds sparsification, pruning, and symbolic snapping to make learned structures human-readable. Across synthetic regression tasks, special functions, PDE solving, knot-theory discovery, and condensed-matter localization, KANs often achieve better accuracy-parameter tradeoffs than MLPs while exposing compact function decompositions that scientists can inspect. The paper's main claim is that KANs are especially attractive for small-scale AI-for-science settings where accuracy and interpretability matter more than raw training speed.

## Problem & Motivation

The paper starts from two limitations of standard [[multi-layer-perceptron]] models: they use many opaque linear weights, and they do not naturally expose the low-dimensional functional structure that many scientific problems may contain. The authors argue that the classical Kolmogorov-Arnold representation suggests a different design principle, where multivariate mappings are built from compositions of univariate functions and addition. Their goal is therefore to construct a practical deep-learning architecture that can exploit compositional structure, achieve better approximation and scaling behavior on structured tasks, and remain inspectable enough to support human-guided scientific discovery instead of only black-box prediction.

## Method

- **KAN layer**: a network with shape `[n_0, n_1, ..., n_L]` uses edge functions `\phi_{l,j,i}` instead of scalar weights, and updates each node by `x_{l+1,j} = \sum_i \phi_{l,j,i}(x_{l,i})`.
- **Edge parameterization**: each activation is `\phi(x) = w_b b(x) + w_s spline(x)`, with residual base function `b(x) = SiLU(x)` and spline term `spline(x) = \sum_i c_i B_i(x)`.
- **Spline setup**: KANs typically use cubic splines with order `k = 3` on `G` grid intervals; for width `N` and depth `L`, the parameter count scales as `O(N^2 L (G + k)) \sim O(N^2 L G)`.
- **Initialization and adaptive grids**: the spline branch starts near zero with `w_s = 1`, the residual branch uses Xavier-style initialization for `w_b`, and each spline grid is updated online so activation support tracks the evolving input range.
- **Approximation theory**: under the assumption that the target admits a smooth KAN representation, the paper proves `||f - \hat{f}_G||_{C^m} \le C G^{-k-1+m}` for `0 \le m \le k`, arguing that approximation error can depend on spline resolution rather than ambient dimension.
- **Grid extension**: a coarse spline is transferred to a finer grid by least-squares fitting `f_fine(x)` to `f_coarse(x)`; in experiments, the authors usually increase `G` every `200` LBFGS steps instead of restarting training from scratch.
- **Sparsification and pruning**: training minimizes `\ell_total = \ell_pred + \lambda(\mu_1 \sum_l |\Phi_l|_1 + \mu_2 \sum_l S(\Phi_l))` with `\mu_1 = \mu_2 = 1`, then prunes nodes whose incoming and outgoing importance stay below `\theta = 10^{-2}`.
- **Symbolification**: learned edge functions can be snapped to affine-wrapped symbolic forms `c f(a x + b) + d`, enabling formula extraction with SymPy after pruning and retraining.
- **Evaluation protocol**: the paper compares KANs to MLPs on toy regression, `15` special functions, Feynman datasets, Poisson-equation [[physics-informed-neural-network]] solving, [[continual-learning]], knot-theory discovery, and mobility-edge discovery in condensed matter physics.

## Key Results

- On five toy functions with known smooth KA structure, KANs nearly follow the predicted `\alpha = 4` scaling law, while MLP baselines struggle even to match the slower `\alpha = 4 / d` trend and plateau quickly.
- In the grid-extension toy task `f(x,y) = exp(sin(\pi x) + y^2)`, the interpolation threshold for a `[2,5,1]` KAN is predicted at `G \approx 1000 / 15 \approx 67` and is observed empirically around `G \sim 50`.
- For the Poisson PINN benchmark, a 2-layer width-10 KAN reaches about `10^-7` MSE with roughly `10^2` parameters, versus about `10^-5` for a 4-layer width-100 MLP with roughly `10^4` parameters.
- On knot-signature classification, a `[17,1,14]` KAN with `G = 3`, `k = 3`, and about `2 x 10^2` parameters achieves `81.6%` test accuracy, exceeding a 4-layer width-300 MLP at `78.0%` with about `3 x 10^5` parameters.
- For mobility-edge discovery, KAN reaches `98.9%` accuracy on GAAM after symbolic snapping and `97.7%` on MAAM after pruning plus manual symbolic simplification, while still returning interpretable formulas.

## Limitations

- The paper states that KANs are typically about `10x` slower to train than MLPs with the same parameter count because edge-wise activations reduce batching efficiency.
- Theoretical guarantees only cover functions that already admit sufficiently smooth KAN-style decompositions; a generalized deep Kolmogorov-Arnold theorem is still missing.
- Most empirical evidence is on small-scale AI-for-science problems, toy datasets, or highly structured formula-discovery settings, so transfer to large-scale mainstream benchmarks remains unclear.
- Architecture search, optimization choices, and symbolic libraries are only partially explored; several strong results still rely on manual pruning choices or human-provided inductive bias.
- The continual-learning advantage is demonstrated only on a simple one-dimensional five-peak task, and the notion of locality in high-dimensional settings is left open.
