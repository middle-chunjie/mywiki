## Summary

The paper presents Agentic Reasoning, a tool-augmented inference framework that lets a reasoning LLM interleave its chain of thought with calls to a Web-Search agent, a Coding agent, and a structured memory agent named Mind-Map. The central design is to offload retrieval, computation, and long-horizon state tracking to specialized agents while preserving coherence through a knowledge graph built from the current reasoning context. With DeepSeek-R1 as the main reasoner, the framework improves Humanity's Last Exam from `9.4%` to `23.8%`, reaches `81.2%` on GPQA, and attains `66.13%` average on GAIA, approaching or surpassing several stronger proprietary systems on specific slices. The paper's main claim is that a small, carefully selected toolbox outperforms both plain reasoning and larger, less targeted tool collections.

## Problem & Motivation

The paper targets open-ended, knowledge-intensive reasoning tasks where pure chain-of-thought is insufficient because the model must repeatedly verify facts, search for recent information, run computations, and preserve complex logical dependencies across long reasoning traces. The authors argue that techniques successful in math and code do not transfer cleanly to domains such as medicine, finance, law, or expert-level QA, where the bottleneck is not only deduction but also evidence gathering, context management, and careful synthesis. Their motivation is to give a strong reasoning model external tools that behave more like human research aids.

## Method

- **Overall pipeline**: a base reasoning model emits special tool-call tokens for `web-search`, `coding`, or `mind-map` requests, pauses the main reasoning chain, sends the current query plus reasoning context to the corresponding agent, then reinserts the returned snippet before continuing inference.
- **Mind-Map agent**: the framework converts the evolving reasoning trace into a [[knowledge-graph]] by extracting entities and semantic relations, then applies [[community-detection]] to cluster the graph and summarize each cluster. The model can also query the graph through [[graph-rag]] when it loses track of earlier facts or logical relations.
- **Web-Search agent**: the search pipeline performs [[query-breakdown]] on the original request using Mind-Map context, submits the refined queries to [[bing]], reranks retrieved pages with a separate model, and runs [[retrieval-augmented-generation]] over pages whose relevance scores exceed `0.7`.
- **Iterative search control**: the agent computes the average relevance of the top `10` pages and, if the average stays below `0.7`, reformulates the search query and retries, with at most `3` iterations. It initially retrieves the top `20` pages from the search engine.
- **Coding agent**: instead of forcing the reasoner to write and execute code itself, a separate [[coding-agent]] receives the user query plus reasoning context, generates code with [[claude-3-5-sonnet]], executes it in `Python 3.11`, and returns a natural-language result for the main chain.
- **Implementation defaults**: the authors use [[deepseek-r1]] as the default reasoner and [[deepseek-v3]] for query breakdown, RAG synthesis, knowledge-graph construction, and graph retrieval, with `max_tokens = 32768`, `temperature = 0.7`, `top_p = 0.8`, `top_k = 20`, and `repetition_penalty = 1.05`. Results are reported as `pass@1`.

## Key Results

- **Humanity's Last Exam**: Agentic Reasoning with [[deepseek-r1]] scores `23.8%`, up from the base model's `9.4%` for a `+14.4` point gain, and narrows the gap to OpenAI Deep Research (`26.6%`) to `2.8` points.
- **GPQA**: the DeepSeek-R1 variant reaches `81.2%` overall, versus `71.5%` for base DeepSeek-R1, `74.6%` for Search-O1 with DeepSeek-R1, and `79.7%` for proprietary `o3-mini-high`.
- **GAIA**: the method achieves `74.36 / 69.21 / 45.46` on Levels `1/2/3`, for `66.13%` average, outperforming h2oGPTe (`65.12%`) and staying within `1.23` points of OpenAI Deep Research (`67.36%`).
- **Deep research generation**: on FreshWiki-style article generation, the system reports `54.10` ROUGE-1, `19.62` ROUGE-L, and `18.77` entity recall, exceeding STORM's `47.93 / 17.42 / 15.43`.
- **Human evaluation**: on `56` expert-authored deep-research questions, the system scores `3.7` interest, `4.6` organization, `4.2` relevance, and `4.1` coverage, outperforming Gemini Deep Research (`2.7 / 2.5 / 2.3 / 3.0`).
- **Ablations**: replacing Mind-Map with no memory gives `46.18` GAIA average, while Mind-Map reaches `66.13`; using direct API calling instead of agentic coding/search yields `47.18`, versus `66.13` for the agent-based design.

## Limitations

- The method incurs substantial latency and compute overhead because web search, code execution, and memory construction are invoked sequentially during inference.
- The framework relies heavily on externally retrieved evidence, but the paper does not provide a strong source-verification module, making the system vulnerable to noisy or misleading web content.
- Mind-Map improves memory structure, but final decisions are still LLM-mediated, so hallucinations can propagate across multi-step reasoning and tool calls.
- The strongest comparisons are against systems built on different underlying reasoning models, so some of the remaining performance gap may reflect base-model differences rather than the agentic framework alone.
