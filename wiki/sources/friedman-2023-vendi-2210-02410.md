## Summary

Friedman and Dieng propose the Vendi Score, a general-purpose diversity metric defined as the exponential of the Shannon entropy of the eigenvalues of a normalized similarity matrix. The key idea is to measure the effective number of distinct elements in a sample without requiring a reference dataset, labels, or a model-defined probability distribution. Because the metric only needs a user-chosen positive semidefinite similarity function, it applies across molecules, images, text decoding, and dataset auditing. The paper formalizes core properties such as effective-number semantics, partitioning, and invariance to duplicates, then shows empirically that Vendi Score often reveals diversity failures that average-pairwise metrics such as IntDiv or simple mode counts miss.

## Problem & Motivation

The paper targets a recurring evaluation gap in machine learning: diversity matters in generative modeling, decoding, and dataset curation, but the available metrics are usually tied to a reference distribution, pretrained classifier, labeled ontology, or domain-specific feature design. These constraints make them hard to reuse across modalities and often cause them to miss internal redundancy such as duplicate clusters or correlated attributes. The authors therefore seek a reference-free metric that depends only on the sample itself plus a user-specified notion of similarity, while still remaining interpretable as an "effective number" of distinct items.

## Method

- Given samples `x_1, ..., x_n` and a positive semidefinite similarity function `k` with `k(x, x) = 1`, build the kernel matrix `K` where `K_{ij} = k(x_i, x_j)`.
- Let `lambda_1, ..., lambda_n` be the eigenvalues of `K / n`. Define the Vendi Score as `VS_k(x_1, ..., x_n) = exp(-sum_i lambda_i log lambda_i)`.
- The metric is equivalently `VS_k = exp(-tr((K / n) log(K / n)))`, making it the exponential of the von Neumann entropy of the normalized similarity matrix.
- The paper proves four main properties: the score is an effective number (`n` when all items are mutually dissimilar, `1` when all are identical), is unchanged by merging exact duplicates, obeys a partitioning law `exp(H(p_1, ..., p_m)) prod_i VS_k(S_i)^{p_i}` for mutually dissimilar groups, and is permutation-invariant.
- The score is closely related to matrix effective rank: with an embedding matrix `X` and inner-product similarity, the nonzero eigenvalues of the kernel and covariance views coincide, so diversity can be computed from low-dimensional features when `d << n`.
- Exact computation from an `n x n` kernel costs `O(n^3)` via eigendecomposition; with explicit `d`-dimensional embeddings it can be reduced to `O(d^2 n + d^3)`, and the paper suggests Nystrom approximation when only the full kernel is available.
- The paper compares against Internal Diversity `IntDiv(x_1, ..., x_n) = 1 - (1 / n^2) sum_{i,j} k(x_i, x_j)` and argues that mean pairwise similarity fails to capture feature correlations and dense duplicate clusters.
- Experimental instantiations use task-specific similarities: an RBF kernel for synthetic mixtures, Morgan fingerprint similarity with radius `2` for molecules, a probability-product kernel `k(x, x') = sum_y p(y | x)^{1/2} p(y | x')^{1/2}` for Stacked-MNIST, cosine similarity over `2048`-dimensional Inception embeddings for image generators, and an averaged `1`- to `4`-gram overlap kernel for MS-COCO captions.
- Evaluation settings are concrete and reproducible: `2,500` valid molecules per molecular model, `10,000` generated samples for Stacked-MNIST checkpoints, `50,000` generated images per image model, beam search with `beam_size = 5`, and diverse beam search with `beam_size = 10`, `group_size = 10`, `return_sequences = 5`, and `gamma in {0.2, 0.6, 0.8}`.

## Key Results

- On molecular generation, Vendi Score sharply disagrees with IntDiv on the HMM model: `IntDiv = 0.871` suggests high diversity, but `VS = 250.9` is far below the original dataset's `403.9` and the combinatorial model's `536.9`, correctly exposing duplicate-heavy samples.
- On Stacked-MNIST mode-collapse evaluation, both PresGAN and self-conditioned GAN achieve `nom = 1000`, but Vendi Score separates them: self-conditioned GAN `VS = 746.7`, PresGAN `VS = 866.6`, original data `VS = 943.7`.
- On CIFAR-10 image generation, the original dataset scores `VS = 19.50`, while the strongest generated model (IDDPM) reaches `16.86`; on ImageNet `64 x 64`, the original dataset is `43.93` and IDDPM is `24.28`, showing persistent diversity gaps even for strong generators.
- On LSUN Cat, ADM attains strong reference-based scores (`Precision = 0.63`, `Recall = 0.52`) but only `VS = 13.09` versus the original dataset's `15.12`, indicating residual internal redundancy despite good quality and coverage.
- In caption decoding, higher diverse-beam-search penalties monotonically increase diversity while lowering BLEU: beam search gives `BLEU = 0.27` and n-gram diversity `0.42`, whereas larger penalties reduce BLEU to `0.25`, `0.22`, `0.21` while raising n-gram diversity to `0.49`, `0.63`, `0.68`; human captions reach `VS = 4.88`.

## Limitations

- Because the metric is reference-free, a set of low-quality but mutually dissimilar samples, including random noise, can achieve a high score; the paper explicitly recommends pairing it with a quality metric.
- The score inherits the inductive bias of the chosen similarity function; an over-sensitive, under-sensitive, or semantically biased kernel can distort conclusions about diversity.
- Exact computation can be expensive when no low-dimensional embedding representation is available, since naive eigendecomposition scales cubically in the sample size.
- Absolute values are not comparable across different similarity functions, so cross-domain or cross-kernel comparisons must be interpreted carefully.
