## Option A — Zotero Advanced Search (recommended, ~30 seconds)

Opens everything in one filtered view. Covers these 45 plus ~60 other orphan PDFs not yet imported here.

1. Open Zotero
2. Menu: Edit → Advanced Search (Cmd+Shift+F on macOS)
3. Match: **All** of the following
4. Add condition: `Item Type` · `is` · `Attachment`
5. Add condition: `Creator` · `does not contain` · (leave empty — filters to items with no author)
6. (Optional) Save as search named `needs-metadata`
7. Results pane shows all orphan PDFs — select all (Cmd+A) → right-click → **Retrieve Metadata**

Zotero will query arxiv/CrossRef/Google Scholar on each PDF's content and auto-create parent items with author/year/venue. After completion, re-export CSL JSON and re-run `/batch-import` to refresh `raw/papers/` slugs.
