## Description

The Java Code Summarization Benchmark is one of the two evaluation datasets used in [[guo-2022-modeling]]. The paper follows the standard split from prior work with `69,708/8,714/8,714` train/validation/test examples.

## Key Contributions

- Serves as the primary benchmark for ablation and model-size analysis in the paper.
- Shows CODESCRIBE reaching `49.19` BLEU, `32.27` METEOR, and `59.59` ROUGE-L.
