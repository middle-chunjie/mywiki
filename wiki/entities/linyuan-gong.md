## Description

Linyuan Gong is a researcher on code large language models and the first author of [[gong-2024-evaluation-2403-04814]], which introduces the SAFIM benchmark for syntax-aware code infilling evaluation.

## Key Contributions

- Co-introduces the SAFIM benchmark for algorithmic block, control-flow, and API-call completion.
- Helps analyze how FIM pretraining, prompt choice, and syntax-aware truncation affect code-LLM evaluation.
