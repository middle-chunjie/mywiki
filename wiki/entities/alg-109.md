## Description

ALG-109 is the algorithm-classification dataset introduced in [[long-2022-multiview]]. It contains `11,913` C/C++ programs collected from CSDN and annotated with `109` algorithm and data-structure labels for multi-label evaluation.

## Key Contributions

- Provides a more realistic and more difficult benchmark than [[poj-104]] for [[algorithm-detection]].
- Supports multi-label evaluation with micro-F1, exact match, and Hamming loss.
- Includes average program length `94.27` lines and average label cardinality `1.94`.
