## Description

Philip S. Yu is a co-author of [[wan-2019-multimodal-1909-13516]] and is affiliated with the University of Illinois at Chicago and the Institute for Data Science, Tsinghua University. He is part of the collaboration proposing MMAN for semantic source code retrieval.

## Key Contributions

- Co-authored the unified formulation that aligns code and natural-language descriptions in a shared retrieval space.
- Contributed to the paper's positioning against prior neural code search baselines.
