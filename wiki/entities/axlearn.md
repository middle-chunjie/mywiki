## Description

AXLearn is Apple's open-source deep-learning framework and is the training stack explicitly used for all models in [[busbridge-2025-distillation-2502-08606]].

## Key Contributions

- Serves as the implementation framework for the paper's transformer teachers and students.
- Supports the large-scale controlled experiments used to fit the distillation scaling law.
