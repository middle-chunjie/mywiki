## Description

OpenNMT is an open-source neural machine translation framework cited by [[ranathunga-2021-neural-2106-15115]] as an accessibility enabler for low-resource NMT research.

## Key Contributions

- Lowers the engineering barrier for building and adapting NMT baselines across new languages.
- Illustrates the paper's argument that open-source tooling accelerates research activity in under-resourced settings.
