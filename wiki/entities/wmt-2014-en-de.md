## Description

Standard machine-translation benchmark from WMT 2014 for English→German translation, containing about 4.5M sentence pairs. Served as the primary evaluation setting for the Transformer.

## Key Contributions

- Reference benchmark on which [[vaswani-2017-attention-1706-03762]] established Transformer's state-of-the-art BLEU of 28.4 (big model).
- newstest2013 is used as the development set (Table 3 ablations); newstest2014 as the test set.
