## Description

Yuanhan Zhang is a researcher affiliated with Nanyang Technological University (NTU), Singapore, working on large vision models, visual in-context learning, and multimodal AI.

## Key Contributions

- First author of [[zhang-nd-what]], the first comprehensive study on in-context example selection for visual in-context learning.
- Proposed the supervised prompt retrieval (SupPR) method that fine-tunes a feature extractor via contrastive learning to maximize visual in-context learning performance.
