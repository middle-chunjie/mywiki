## Description

Gopher is the `280B`-parameter DeepMind language model used in [[hoffmann-2022-training-2203-15556]] as the main large-scale baseline for compute-optimality comparisons.

## Key Contributions

- Serves as the under-trained baseline showing that a much smaller model can outperform it at the same pretraining FLOP budget.
- Anchors the paper's argument that many frontier dense LLMs emphasized parameter count more than token count.
