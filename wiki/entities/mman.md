## Description

MMAN is a state-of-the-art code-search baseline referenced in [[sun-2022-code-2202-08029]]. It combines token, AST, and CFG encoders to model semantic source-code retrieval.

## Key Contributions

- Provides the strongest baseline in the paper's main comparison table.
- Motivates TranCS's argument that AST and CFG structure alone are insufficient for faithful code semantics.
