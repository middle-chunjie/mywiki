## Description

FLORES is the evaluation dataset suite cited in [[ranathunga-2021-neural-2106-15115]] as an example of benchmark release increasing research activity for low-resource languages such as Sinhala and Nepali.

## Key Contributions

- Demonstrates how shared evaluation resources can raise visibility for under-studied language pairs.
- Supports the paper's claim that dataset availability strongly predicts low-resource NMT activity.
