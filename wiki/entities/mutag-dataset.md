## Description

MUTAG is a molecular graph classification dataset from TUDataset that the paper Multi-Scale Subgraph Contrastive Learning uses in both its semantic-similarity analysis and unsupervised evaluation.

## Key Contributions

- Serves as one of the four datasets for analyzing how subgraph semantic similarity changes with scale.
- Reports unsupervised classification accuracy of `89.68 +- 0.57` for MSSGCL.
