## Description

OpenHands is the agent scaffolding used with Claude-based agents on CUA-SHADE-Arena in [[kale-2026-reliable-2508-19461]]. In this source it is the execution framework for computer-use monitoring experiments.

## Key Contributions

- Serves as the agent scaffolding for the paper's Claude Sonnet 4 computer-use evaluations.
- Connects monitor reliability results to realistic code-editing and VM-based agent workflows.
