## Description

GPT-3 is one of the large parametric baselines referenced in [[borgeaud-2022-improving-2112-04426]] to contextualize RETRO's scaling results. The paper argues that RETRO reaches comparable broad performance while using far fewer parameters.

## Key Contributions

- Acts as a comparison point for the claim that retrieval can substitute for part of parameter scaling.
- Helps frame RETRO as a semi-parametric alternative to very large closed-book language models.
