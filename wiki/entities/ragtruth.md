## Description

RAGTruth is a human-annotated hallucination detection benchmark for [[retrieval-augmented-generation]] used as one of the main evaluation datasets in [[yeh-2026-lumina-2509-21875]].

## Key Contributions

- Supplies span-level hallucination annotations over QA, summarization, and data-to-text tasks.
- Supports Lumina's main AUROC, AUPRC, PCC, robustness, and proxy-detector evaluations.
