## Description

Yutong Xie is a researcher affiliated with the [[university-of-michigan]] and a co-first author of [[long-2022-multiview]]. Xie co-authored the MVG approach for code understanding and algorithm classification.

## Key Contributions

- Co-developed the multi-view graph encoding strategy in MVG.
- Contributed to the evaluation design spanning [[poj-104]] and [[alg-109]].
- Helped frame semantic graph views as an alternative to syntax-dominant code models.
