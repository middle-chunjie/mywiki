## Description

Yueqi Song is a researcher at Carnegie Mellon University and the lead author of [[song-2026-agent-2510-24702]]. In this paper, Song helps introduce Agent Data Protocol as a standardized interface for agent-training data.

## Key Contributions

- Co-proposed ADP as a reusable schema for agent trajectories.
- Helped assemble the `1.3M`-trajectory ADP Dataset V1 from `13` source datasets.
- Evaluated ADP-based fine-tuning across coding, browsing, and tool-use benchmarks.
