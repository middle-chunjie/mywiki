## Description

Ge Zhang is a researcher affiliated with ByteDance Seed and a co-author of ReTool. The paper links Zhang to the design and evaluation of RL-trained strategic tool use in LLMs.

## Key Contributions

- Co-authored the ReTool framework for interleaved code execution during reasoning rollouts.
- Helped report large AIME gains over text-only RL baselines.
