## Description

MIRACL is a multilingual retrieval benchmark covering `18` languages, used in [[wang-2025-jinarerankerv-2509-25085]] to measure cross-lingual reranking quality. It is the paper's main multilingual evaluation suite.

## Key Contributions

- Provides the multilingual nDCG@10 benchmark on which jina-reranker-v3 reports `66.83`.
- Tests cross-lingual generalization across diverse language families rather than English-only retrieval.
