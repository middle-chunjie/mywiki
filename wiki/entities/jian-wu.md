## Description

Jian Wu is a co-author of [[wan-2019-multimodal-1909-13516]] and is affiliated with Zhejiang University. The paper lists Wu among the researchers building and evaluating the MMAN code retrieval model on a large C corpus.

## Key Contributions

- Helped curate the experimental dataset and evaluation setup for semantic code retrieval.
- Co-authored the empirical comparison against CodeHow and DeepCS.
