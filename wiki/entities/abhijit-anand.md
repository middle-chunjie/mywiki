## Description

Abhijit Anand is a researcher at [[l3s-research-center]] and a co-author of [[anand-2024-understanding-2408-17103]], which introduces the DL-MIA intent-ranking dataset.

## Key Contributions

- Co-authored the paper that defines the DL-MIA dataset construction and evaluation pipeline.
- Helped frame user intent understanding as an evaluation problem for web ranking.
