## Description

WebCPM is the Chinese web-enhanced question answering dataset used as the base source for constructing synthetic samples in [[xu-2024-facerag-2407-01080]].

## Key Contributions

- Provides seed question-answer materials from which Face4RAG derives synthetic positive and negative examples.
- Serves as the starting point for prompt-based generation of multiple factual inconsistency error types.
