## Description

Weimin Xiong is the first author of [[xiong-2025-mpo-2503-02682]]. This staging page is conservatively disambiguated from [[weimin-xiong]] because the available paper markdown omits the affiliation header; the `-pku` suffix reflects the paper's broader co-author cluster around Sujian Li and related Peking University-linked work, and should be re-verified against the original PDF header.

## Key Contributions

- Co-proposed MPO, which trains a detachable meta planner to generate explicit high-level plans for LLM agents.
- Helped define the Monte Carlo execution-based scoring loop that turns downstream task success into plan preferences for DPO.
- Co-authored the cross-benchmark evaluation on ScienceWorld, ALFWorld, and WebShop.
