## Description

Weimin Xiong is the first author of [[unknown-nd-program-2310-05727]] and is affiliated with [[tencent]] and [[peking-university]] in the paper header. The paper credits him on the empirical study of how code LLMs generate tests and how those tests can improve synthesis.

## Key Contributions

- Co-designed the comparative evaluation of `11` code LLMs across four program-testing settings.
- Helped formulate the deduplicated pass-rate and branch-coverage analysis used to characterize testing ability.
- Co-proposed the self-generated and rank-weighted test-case strategy for improving CodeT-style program synthesis.
