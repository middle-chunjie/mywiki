## Description

Yujia Qin is a researcher affiliated with ByteDance Seed and a co-author of ReTool. In this paper, Qin contributes to learning when and how an LLM should call a code interpreter while solving math problems.

## Key Contributions

- Co-authored ReTool's outcome-driven RL setup for tool use.
- Helped frame tool invocation timing and code complexity as behaviors that can emerge from final-answer supervision.
