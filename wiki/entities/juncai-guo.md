## Description

Juncai Guo is the first author of [[guo-2022-modeling]] and was affiliated with Wuhan University at the time of publication. In this paper he helps introduce CODESCRIBE for source code summarization with triplet-position-aware AST modeling.

## Key Contributions

- Co-designed the triplet position representation for ordered AST structure.
- Co-authored the GraphSAGE-plus-Transformer architecture and its evaluation on Java and Python benchmarks.
