## Description

Yingwei Ma is a researcher affiliated with the National University of Defense Technology and Peng Cheng Laboratory. In this paper, he is the co-first author leading the empirical study of when code data improves LLM reasoning.

## Key Contributions

- Co-authored the comparison between code injection during pre-training and instruction tuning.
- Helped analyze how code data affects general reasoning, code reasoning, and Chain-of-Thought performance.
