## Description

Pyre Infer is Meta's static type inference tool for Python and is used as a major static-analysis baseline in the HiTYPER paper.

## Key Contributions

- Acts as a comparison point for exact-match precision and number of correct inferred annotations.
- Demonstrates the gap between conventional static inference coverage and HiTYPER's TDG-based approach.
