## Description

Gymnasium is the reinforcement-learning environment library whose implementations and documentation were adapted to build the Code World Models Benchmark in [[dainese-2024-generating-2405-15383]].

## Key Contributions

- Provides the source environments and textual descriptions underlying the 18-task CWMB benchmark.
- Supplies environment semantics that are converted into language descriptions for LLM-guided code synthesis.
