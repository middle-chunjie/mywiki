## Description

CommonsenseQA (CSQA) is a multiple-choice commonsense reasoning benchmark with `1,221` test questions (average `27.8` words), requiring models to reason about prior world knowledge (Talmor et al., 2019). Questions target complex semantics and implicit commonsense inferences.

## Key Contributions

- One of ten benchmark tasks used to evaluate Auto-CoT; Auto-CoT achieves `74.4%` accuracy vs. `73.5%` for Manual-CoT using GPT-3 with `k = 7` demonstrations.
- Represents the commonsense reasoning category in the Auto-CoT evaluation suite alongside StrategyQA.
