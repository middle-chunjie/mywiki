## Description

pykan is the open-source codebase referenced by [[liu-2024-kan-2404-19756]] for implementing and experimenting with Kolmogorov-Arnold Networks. In this source it provides the practical interface for training, pruning, plotting, and symbolic manipulation of KAN models.

## Key Contributions

- Provides a public implementation of KAN training and visualization.
- Exposes operations such as pruning, symbolic fixing, automatic symbolic suggestions, and formula export that support the paper's interpretability claims.
- Serves as the practical artifact through which the paper's KAN workflow can be reproduced.
