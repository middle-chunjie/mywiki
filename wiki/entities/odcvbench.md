## Description

ODCVBench is cited in [[huang-2025-audited-2512-23760]] as a benchmark for outcome-driven constraint violations in multi-step environments, where agents may be rewarded for achieving outcomes while breaking rules.

## Key Contributions

- Provides the paper's main reference point for evaluating optimization-induced constraint violations.
- Supports ASG-SI's threat model and its emphasis on explicit constraint checks inside evidence bundles.
