## Description

Gauthier Guinet is a researcher affiliated with [[amazon-science]] and a co-author of [[guinet-2024-automated-2405-13622]]. In this source he helps frame automated, task-specific evaluation for RAG pipelines through synthetic exams and hierarchical IRT analysis.

## Key Contributions

- Co-authored the exam-based evaluation framework for retrieval-augmented language models.
- Helped connect question quality, retriever choice, and model ability decomposition in the reported benchmark.
- Contributed to the accompanying [[auto-rag-eval]] implementation released with the paper.
