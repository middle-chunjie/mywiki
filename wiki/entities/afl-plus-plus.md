## Description

AFL++ is the fuzzing engine used in [[unknown-nd-code-2309-09980]] to generate diverse program inputs for collecting execution-derived test cases. It is the core tool behind the paper's dynamic-information data pipeline.

## Key Contributions

- Supplies fuzzed inputs that help cover diverse logic paths in the pre-training corpus.
- Enables the construction of input-output pairs used for dynamic information matching and distillation.
