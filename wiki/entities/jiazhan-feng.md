## Description

Jiazhan Feng is a researcher affiliated with ByteDance Seed and a co-author of ReTool. In this paper, Feng studies how reinforcement learning can teach LLMs to invoke a code interpreter strategically during mathematical reasoning.

## Key Contributions

- Co-authored ReTool, a framework that interleaves code execution with RL rollouts for LLM reasoning.
- Helped analyze how code usage timing, code complexity, and self-correction emerge during tool-augmented training.
