## Description

OpenAI Codex is the large language model evaluated in [[geng-2024-large]]. The paper describes it as a GPT-3-based decoder model trained on large-scale code and comment data.

## Key Contributions

- Serves as the zero-shot, one-shot, and few-shot engine for all main experiments in the paper.
- Establishes a stronger baseline than the supervised DOME system under well-designed prompting, retrieval, and reranking.
