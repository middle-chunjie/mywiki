## Description

CodeActAgent is the open-source agent model family trained in [[wang-2024-executable-2402-01030]] from Llama-2 and Mistral backbones. It is designed to interact with environments through executable Python and to collaborate with users via natural language.

## Key Contributions

- Demonstrates that CodeAct-style instruction tuning can improve open-source agent performance.
- Integrates naturally with Python packages for tasks such as data processing, modeling, and visualization.
- Shows self-debug behavior by using execution errors to revise later actions.
