## Description

Sida Wang is a co-author of [[gong-2024-evaluation-2403-04814]] and contributes to the paper's study of syntax-aware code fill-in-the-middle evaluation for code LLMs.

## Key Contributions

- Co-develops the SAFIM benchmark and its multilingual task splits.
- Contributes to the empirical comparison of FIM-pretrained and non-FIM-pretrained code models.
