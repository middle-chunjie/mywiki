## Description

Harbor Framework is the containerized evaluation framework on which SkillsBench is built. It provides the task structure, execution environment, and isolation model used for reproducible agent benchmarking.

## Key Contributions

- Supplies the containerized task abstraction used by SkillsBench.
- Supports isolated execution with task-specific environments, oracle solutions, and deterministic verifiers.
