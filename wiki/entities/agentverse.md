## Description

AgentVerse is a human-designed multi-agent framework used as a baseline in [[yuan-2024-evoagent-2406-14228]]. The paper treats it as a strong comparison point for automatic agent generation because it already supports expert recruitment and collaborative decision making.

## Key Contributions

- Serves as one of the main handcrafted multi-agent baselines compared against EvoAgent.
- Illustrates the paper's argument that fixed framework design limits scalability relative to evolutionary agent generation.
