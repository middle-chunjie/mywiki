## Description

Niels Mündler is a researcher affiliated with ETH Zurich and the lead author of the 2024 SWT-Bench paper on LLM-based software test generation. In this source he helps frame issue-reproducing tests as a benchmarkable task for code agents.

## Key Contributions

- Co-created SWT-Bench by transforming SWE-Bench-style issue/patch pairs into a test-generation benchmark.
- Evaluated zero-shot LLM baselines and code agents for issue-reproducing tests.
