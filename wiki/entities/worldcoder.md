## Description

WorldCoder is the main LLM-based baseline compared against GIF-MCTS in [[dainese-2024-generating-2405-15383]] for synthesizing code world models from environment feedback.

## Key Contributions

- Serves as the paper's strongest iterative code-generation baseline on CWMB and RTFM.
- Motivates the paper's argument that MCTS-style search can outperform Thompson-sampling-based refinement in this setting.
