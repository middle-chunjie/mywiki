## Description

DuRecDial is a public [[knowledge-grounded-dialogue]] benchmark with recommendation-oriented multi-turn interactions. [[wang-2024-improving-2401-04361]] uses it as the third core dataset for evaluating EnCo.

## Key Contributions

- Contains about `145K` samples built from `10.2K` dialogues.
- Includes `21.8K` entities, `454` relations, and `222.2K` knowledge triples.
- Shows that EnCo generalizes beyond KdConv and DuConv to another KGD benchmark.
