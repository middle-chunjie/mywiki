## Description

CoNaLa is an English Stack Overflow-derived natural-language-to-code dataset used as a source resource in [[wang-2023-executionbased-2212-10481]]. ODEX extends CoNaLa by adding executable context and human-written tests.

## Key Contributions

- Supplies practical coding queries and canonical code snippets harvested from Stack Overflow.
- Serves as one of the two raw data sources from which ODEX is constructed.
