## Description

Jielu Zhang is a PhD student at the University of Georgia and co-first author (equal contribution) of the Img2Loc paper on multimodal image geolocation.

## Key Contributions

- Co-designed the Img2Loc system combining CLIP-based retrieval with LMM prompt augmentation.
- Previous work includes Text2Seg for remote sensing image semantic segmentation via visual foundation models.
