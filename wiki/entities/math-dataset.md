## Description

MATH is a competition-level mathematical reasoning benchmark used as a primary training and evaluation dataset in [[gou-2024-tora-2309-17452]]. The paper uses it both for GPT-4 trajectory synthesis and for measuring in-domain reasoning performance.

## Key Contributions

- Serves as one of the two core datasets used to build TORA-CORPUS.
- Provides the main high-difficulty benchmark where ToRA reports its headline gains, including `50.8%` for ToRA-CODE-34B.
