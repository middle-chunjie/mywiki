## Description

Amazon Web Services (AWS) is the cloud computing division of Amazon, which also hosts an applied science research organization that publishes work in machine learning, NLP, and AI. Authors Aston Zhang, Mu Li, and Alex Smola are affiliated with AWS in the Auto-CoT paper.

## Key Contributions

- Contributed researchers (Aston Zhang, Mu Li, Alex Smola) to the Auto-CoT automatic chain-of-thought prompting work.
- Hosts the amazon-research GitHub organization where the Auto-CoT code was released.
