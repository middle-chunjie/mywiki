## Description

The Python Code Summarization Benchmark is the second evaluation dataset used in [[guo-2022-modeling]]. The paper follows the established split with `55,538/18,505/18,502` train/validation/test examples.

## Key Contributions

- Validates that CODESCRIBE generalizes beyond Java to a second programming language benchmark.
- Shows CODESCRIBE reaching `35.11` BLEU, `23.48` METEOR, and `50.46` ROUGE-L.
