## Description

Georgios Gousios is a researcher at [[delft-university-of-technology]] and a co-author of [[izadi-2022-codefill-2202-06689]]. In this paper, he contributes to the dataset construction, system design, and evaluation methodology for learning-based code completion.

## Key Contributions

- Co-authored the large-scale Python dataset and the deduplication/evaluation setup used by CodeFill.
- Helped position CodeFill against both grammar-based and neural autocompletion baselines.
