## Description

JavaScript150k is the JavaScript benchmark dataset used in [[peng-nd-rethinking]] to test whether the proposed tree positional encoding transfers across programming languages and tasks.

## Key Contributions

- Provides `186k / 23k / 93k` train/validation/test functions for summarization experiments.
- Provides `270k / 32k / 220k` AST chunks for completion experiments, enabling cross-language comparison with Python150k.
