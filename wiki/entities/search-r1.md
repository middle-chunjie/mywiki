## Description

Search-R1 is a single-tool reasoning baseline referenced throughout the paper. It is also the model used in the preliminary entropy study that motivates Tool-Light's entropy-guided sampling design.

## Key Contributions

- Supplies the multi-dataset trajectories used to study how entropy changes around tool calls.
- Acts as a competitive single-tool baseline with `45.6` average performance in the main table.
