## Description

Open-source deep-learning library released by Google for training sequence models, including the reference Transformer implementation. The code used to train and evaluate the models in [[vaswani-2017-attention-1706-03762]] is hosted at https://github.com/tensorflow/tensor2tensor.

## Key Contributions

- First public reference implementation of the Transformer.
- Standardized the Transformer training recipe (warmup schedule, label smoothing, checkpoint averaging) for reproduction.
