## Description

MP-16 (MediaEval Placing Tasks 2016) is a large-scale dataset of 4.72 million geotagged images sourced from Flickr, assembled for the MediaEval 2016 multimedia evaluation benchmark. In Img2Loc it serves as the reference gallery: all images are encoded with CLIP and stored in a FAISS vector database for nearest-neighbor retrieval.

## Key Contributions

- Provides the 4.72-million-image embedding database used by Img2Loc for retrieval-augmented geolocation.
- Offers near-global geographic coverage at the scale needed to support planet-level nearest-neighbor search.
