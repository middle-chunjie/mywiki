## Description

Chin-Yew Lin is a Microsoft Corporation researcher and co-author of [[jiang-2023-llmlingua-2310-05736]], which studies accelerated LLM inference through prompt compression.

## Key Contributions

- Co-developed the LLMLingua pipeline for preserving task performance under aggressive prompt compression.
- Contributed to the paper's evaluation design and analysis on black-box LLMs.
