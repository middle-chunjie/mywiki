## Description

Federico Mora is a UC Berkeley researcher and the first author of this paper introducing SPEAC for VLPL text-to-code generation. He contributes to the method design, prototype implementation, and evaluation around UCLID5.

## Key Contributions

- Co-introduces SPEAC as an interface-and-repair strategy for very low-resource code generation.
- Helps build and evaluate the Eudoxus prototype for UCLID5.
