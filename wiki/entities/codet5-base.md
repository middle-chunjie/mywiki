## Description

CodeT5-base is an encoder-decoder pretrained code model used both as a strong baseline and as the editor initialization in [[li-2023-skcoder-2302-06144]].

## Key Contributions

- Serves as the strongest direct baseline that SKCoder consistently outperforms on all three datasets.
- Provides pretrained weights for the editor module in the final SKCoder implementation.
