## Description

RULER is a synthetic long-context benchmark with controllable sequence lengths, used in [[yu-2026-memagent-2507-02259]] as the main evaluation suite for extrapolation and out-of-distribution testing.

## Key Contributions

- Provides the primary accuracy table spanning contexts from `7K` to `3.5M` tokens for MemAgent.
- Supplies out-of-distribution tasks such as needle-in-a-haystack, variable tracking, frequent-word extraction, and QA variants.
