## Description

Bing Xiang is a researcher affiliated with [[aws-ai-labs]] and a co-author of [[ding-2023-cocomic-2212-10007]]. In this paper, he contributes to repository-level context modeling for code generation and completion.

## Key Contributions

- Co-authored the CoCoMIC framework and its empirical validation.
- Supported the formulation of cross-file retrieval as a practical complement to autoregressive code modeling.
