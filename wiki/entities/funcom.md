## Description

Funcom is a large Java method summarization dataset used in [[shi-2022-evaluation-2107-07112]]. The paper uses it to study how larger corpora and different split strategies affect neural code summarization performance.

## Key Contributions

- Provides `2,118,419` method-summary pairs from `28,790` projects after filtering.
- Supports controlled experiments on corpus size and project-vs-method splitting.
- Shows that NCS achieves the best reported score on the base FCM setting with `18.07 +- 0.46`.
