## Description

SCodeR is the pre-training framework proposed in [[li-2022-softlabeled-2210-09597]] for learning function-level code representations with soft-labeled contrastive learning, code-comment positives, and AST-based code splitting.

## Key Contributions

- Combines discriminator-derived soft labels with dual-encoder contrastive pre-training.
- Introduces comment-code and ASST-based positive-pair construction.
- Sets new reported state of the art on four downstream code understanding tasks in the paper.
