## Description

Sanket Vaibhav Mehta is a Google DeepMind researcher and co-author of [[kazemi-2025-bigbench-2502-19187]], which introduces [[bbeh]] for more discriminative reasoning evaluation.

## Key Contributions

- Co-authored benchmark tasks that increase reasoning depth, context length, and distractor load.
- Helped evaluate differences between general-purpose and reasoning-specialized models.
