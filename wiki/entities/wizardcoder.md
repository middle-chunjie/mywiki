## Description

WizardCoder is the open-source code large language model family used in [[unknown-nd-codechaintowards-2310-08992]] as a base model for evaluating CodeChain, with experiments spanning `1B` to `34B` parameters. The paper highlights it as the main open-weight comparison against OpenAI models.

## Key Contributions

- Serves as the paper's open-source backbone for testing whether CodeChain transfers beyond proprietary APIs.
- WizardCoder-15B improves from `7.90` to `10.50` pass@1 on APPS and from `1.98` to `2.48` pass@1 on CodeContests under CodeChain.
