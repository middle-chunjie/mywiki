## Description

Nicola Cancedda is a Meta AI Research coauthor of [[schick-2023-toolformer-2302-04761]]. In this work he contributes to a method that teaches a language model to insert API calls only when they reduce future-token prediction loss.

## Key Contributions

- Coauthored the perplexity-based filtering objective that decides which sampled API calls remain in training data.
- Helped validate that tool-augmented finetuning improves zero-shot downstream accuracy without worsening no-tool perplexity.
- Contributed to the study's framing of tool use as selective augmentation rather than unconditional retrieval.
