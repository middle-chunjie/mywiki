## Description

PanGu-Coder is a code generation model evaluated in [[yu-2024-codereval-2302-00288]] on both Python and Java tasks. It is one of the three main baselines used to quantify the gap between standalone and pragmatic code generation.

## Key Contributions

- Achieves stronger CoderEval results than CodeGen on both languages, including Java `Pass@1/5/10 = 25.43/37.39/43.04`.
- Helps show that monolingual or language-specialized pretraining can improve stability under prompt variation.
