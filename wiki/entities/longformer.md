## Description

Longformer is a sparse-attention Transformer model used as a long-context baseline in [[li-2024-chulo-2410-11119]]. The paper compares ChuLo against Longformer on both document classification and token classification tasks.

## Key Contributions

- Serves as a `4096`-token sparse-attention baseline for long-document processing.
- Provides a strong reference point showing where ChuLo's chunk representation is better or roughly tied.
