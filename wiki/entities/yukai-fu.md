## Description

Yukai Fu is a researcher affiliated with the Chinese Academy of Sciences in Shenyang and a co-author of [[li-2024-building]]. He works on retrieval-augmented methods for code intelligence.

## Key Contributions

- Co-authored CONAN, which introduces structure-aware retriever pretraining and dual-view code representation for code assistance.
- Helped evaluate the framework on retrieval benchmarks and multiple code generation tasks.
