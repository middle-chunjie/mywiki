## Description

AGIEval is a bilingual (Chinese/English) benchmark for evaluating foundation models on 20 tasks derived from official, high-standard human exams — including Gaokao, SAT, LSAT, GMAT, GRE, AMC/AIME math competitions, JEC-QA, and civil service exams — totalling 8,062 objective questions. It provides human-level performance baselines (average and top-1%) for each task.

## Key Contributions

- First benchmark to systematically source from officially validated, high-stakes human examinations across multiple languages and cognitive skill dimensions.
- Provides data, code, and all model outputs at https://github.com/ruixiangcui/AGIEval.
- Includes data contamination analysis via timestamped Gaokao subsets.
