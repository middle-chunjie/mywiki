## Description

Beijun Shen is a co-author of [[gu-2018-deep]] and is affiliated with [[shanghai-jiao-tong-university]]. The paper associates Shen with the research effort on cross-domain code search under scarce supervision.

## Key Contributions

- Co-authored the CDCS study on few-shot meta learning for code search.
- Helped validate the method on SQL and Solidity benchmarks.
