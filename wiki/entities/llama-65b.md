## Description

Llama 65B is the base large language model used to instantiate the baseline and all CoVe variants in [[dhuliawala-2023-chainofverification-2309-11495]].

## Key Contributions

- Serves as the shared backbone for few-shot baseline, joint CoVe, two-step CoVe, factored CoVe, and factor+revise.
- Shows measurable gains from inference-time verification without any weight updates or retrieval augmentation.
