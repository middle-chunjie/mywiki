## Description

BANKING is a fine-grained intent detection benchmark for banking dialogue systems, containing 77 intent categories and 13,083 utterances (train: 9,003, validation: 1,000, test: 3,080). It is widely used to evaluate new intent discovery and intent classification methods.

## Key Contributions

- Primary evaluation benchmark for NID in the banking domain; used in MTP-CLNN experiments.
- Fine-grained label space (77 intents) makes clustering challenging and tests model discriminability.
