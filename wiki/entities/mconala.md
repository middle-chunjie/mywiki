## Description

MCoNaLa is the multilingual extension of CoNaLa used as a source dataset in [[wang-2023-executionbased-2212-10481]]. It provides non-English natural-language intents that ODEX turns into executable benchmark items.

## Key Contributions

- Supplies Spanish, Japanese, and Russian coding intents harvested from Stack Overflow.
- Enables ODEX to support multilingual execution-based evaluation rather than English-only benchmarking.
