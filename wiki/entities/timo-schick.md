## Description

Timo Schick is the lead author of [[schick-2023-toolformer-2302-04761]] and is listed with Meta AI Research in the paper. His contribution centers on teaching a pretrained language model to decide autonomously when external tools improve next-token prediction.

## Key Contributions

- Co-introduced Toolformer, a self-supervised pipeline for sampling, filtering, and finetuning on API calls.
- Helped frame tool use as a language-model pretraining problem rather than a task-specific prompting trick.
- Coauthored the empirical analysis showing large zero-shot gains without degrading base perplexity.
