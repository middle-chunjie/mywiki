## Description

Python150k is one of the two benchmark datasets used in [[peng-nd-rethinking]] for evaluating AST-based code summarization and code completion on Python source files collected from GitHub.

## Key Contributions

- Provides the Python benchmark split for both function-name summarization and AST node completion.
- Supplies `523k / 56k / 264k` examples for summarization and `186k / 20k / 100k` chunks for completion in train/validation/test.
