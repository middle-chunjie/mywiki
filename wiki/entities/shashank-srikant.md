## Description

Shashank Srikant is the first author of this paper and is listed with CSAIL, MIT. In this work he helps formulate adversarial program generation through optimized obfuscation and first-order optimization.

## Key Contributions

- Co-develops the joint site-selection and site-perturbation formulation for adversarial programs.
- Evaluates AO, JO, and randomized smoothing on Python and Java program summarization.
