## Description

TopiOCQA is a conversational QA and retrieval benchmark used for normal evaluation in this paper. It contains topic-switching conversational interactions that stress robustness across turns.

## Key Contributions

- Provides a second in-domain benchmark for validating ConvAug.
- Helps reveal that ConvAug improves performance especially on later turns of long conversations.
