## Description

Yun Peng is a researcher at The Chinese University of Hong Kong and the first author of the HiTYPER paper on hybrid type inference for Python.

## Key Contributions

- Co-proposed HiTYPER as a hybrid framework combining static inference and neural type recommendation.
- Contributed to the TDG-based formulation of Python type inference.
- Co-evaluated the method on Typilus and ManyTypes4Py benchmarks.
