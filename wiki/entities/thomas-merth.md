## Description

Thomas Merth is a coauthor of [[merth-2024-superposition-2404-06910]] and works on efficient LLM inference for retrieval-augmented generation. The paper materials suggest an affiliation with [[apple]], where the linked code release was published.

## Key Contributions

- Coauthored the superposition prompting framework for graph-structured RAG prompting.
- Helped validate pruning, caching, and parallelization across multiple open LLM families.
