## Description

The Helpful and Harmless dataset is Anthropic's preference dataset used in [[yuan-2023-rrhf-2304-05302]] for RRHF training, automatic evaluation, and human comparison studies.

## Key Contributions

- Provides chosen and rejected responses for each query, enabling response ranking and reward-model-based evaluation.
- Serves as the main benchmark on which RRHF is compared against PPO, SFT, and provided human responses.
