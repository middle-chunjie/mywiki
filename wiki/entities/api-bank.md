## Description

API-Bank is one of the evaluation datasets used in [[unknown-nd-tool-2508-05152]] for tool retrieval, providing user queries, ground-truth tools, and execution outcomes. The paper uses `311` test samples from API-Bank.

## Key Contributions

- Serves as one of the main retrieval benchmarks for TGR.
- Supports measurement of Recall, NDCG, and Pass Rate under top-`k` retrieval.
