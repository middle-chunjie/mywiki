## Description

Laurent Callot is a researcher affiliated with [[amazon-science]] and a co-author of [[guinet-2024-automated-2405-13622]]. In this source he is linked to the overall evaluation design that compares retrieval strategies, model scale, and in-context learning regimes for RAG.

## Key Contributions

- Co-authored the benchmark and analysis showing retriever choice can outweigh simple LLM scaling.
- Helped position the framework as both predictive evaluation and prescriptive model-selection support.
- Contributed to the open-source release of [[auto-rag-eval]].
