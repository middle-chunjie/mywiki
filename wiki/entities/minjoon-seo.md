## Description

Minjoon Seo is a RouterRetriever co-author and appears in the paper as one of the `1`-affiliated authors together with Hyunji Lee. In the context of this paper, Seo is associated with the design and analysis of retrieval routing over multiple expert embedders.

## Key Contributions

- Co-authored RouterRetriever, a modular dense retrieval method built from a frozen base encoder plus domain-specific LoRA experts.
- Helped evaluate how routing quality changes with the number of experts and how far the method remains from instance-level oracle routing.
