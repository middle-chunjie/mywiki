## Description

CUA-SHADE-Arena is the paper's new benchmark for monitoring computer-use agents, extending [[the-agent-company]] with harmful side tasks. In [[kale-2026-reliable-2508-19461]] it is used to test monitors on more realistic VM-based agent behavior.

## Key Contributions

- Introduces `15` main-side task pairs for computer-use agent monitoring.
- Expands MRT beyond tool-calling environments into virtual-machine interaction settings.
