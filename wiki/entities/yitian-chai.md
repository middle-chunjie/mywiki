## Description

Yitian Chai is the first author of [[gu-2018-deep]] and is affiliated with [[shanghai-jiao-tong-university]]. In this paper Chai contributes to the design and evaluation of cross-domain deep code search with few-shot meta learning.

## Key Contributions

- Co-authored the CDCS method for adapting code search models to low-resource domain-specific languages.
- Helped evaluate cross-domain transfer from Python and Java to Solidity and SQL.
