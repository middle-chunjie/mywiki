## Description

The Agent Company is the benchmark suite that underlies [[cua-shade-arena]] in [[kale-2026-reliable-2508-19461]]. It provides benign computer-use tasks representing realistic workplace jobs for web agents.

## Key Contributions

- Supplies the benign task base from which CUA-SHADE-Arena side-task pairs are constructed.
- Grounds the paper's monitoring experiments in virtual-machine computer-use workflows.
