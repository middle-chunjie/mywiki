## Description

Haozhen Zhang is a researcher at Nanyang Technological University (NTU), Singapore, and the first author of MemSkill. His work focuses on LLM agent memory systems and self-evolving agent learning.

## Key Contributions

- First author and lead researcher of MemSkill, proposing the learnable and evolvable memory skill framework for LLM agents.
- Designed the closed-loop optimization combining RL-based controller training with LLM-guided skill evolution.
