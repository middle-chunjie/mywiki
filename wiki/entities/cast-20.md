## Description

CAsT-20 is a TREC Conversational Assistance Track benchmark used for zero-shot evaluation in this paper. It measures whether a conversational retriever trained elsewhere can generalize to a different conversational search distribution.

## Key Contributions

- Serves as a target benchmark for zero-shot evaluation of ConvAug.
- Shows ConvAug reaching `MRR 45.0` and `NDCG@3 30.7`, outperforming all reported baselines.
