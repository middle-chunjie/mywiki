## Description

Ali TehraniJamsaz is a researcher at Iowa State University and a coauthor of [[tehranijamsaz-2024-coderosetta-2410-20527]]. In this paper he contributes to an unsupervised encoder-decoder approach for translating between general-purpose code and HPC extensions.

## Key Contributions

- Coauthored CodeRosetta, which combines [[masked-language-modeling]], [[abstract-syntax-tree-entity-recognition]], and [[denoising-autoencoding]] for bidirectional code translation.
