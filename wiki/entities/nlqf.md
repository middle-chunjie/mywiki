## Description

NLQF is the Python library released by [[sun-2022-importance-2202-06649]] to operationalize its query-cleaning framework for neural code search datasets.

## Key Contributions

- Packages the rule-based and model-based filtering pipeline as a reusable library.
- Exposes configurable rules, open-source VAE training code, and tunable dividing-point selection.
