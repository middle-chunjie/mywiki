## Description

Hongyi Jin is a researcher affiliated with [[carnegie-mellon-university]] and a co-author of this survey on efficient LLM serving. In this source he contributes to the synthesis of practical systems techniques for large-scale generative inference.

## Key Contributions

- Co-authors the survey's review of memory management, offloading, and serving-system architecture.
- Helps summarize how system-level optimizations interact with algorithmic changes in deployed LLM services.
