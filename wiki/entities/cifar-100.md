## Description

CIFAR-100 is the fine-grained `100`-class benchmark used in [[sener-2018-active-1708-00489]] to stress-test the proposed active learning method.

## Key Contributions

- Provides the hardest image classification setting among the reported experiments.
- Illustrates the paper's observation that performance weakens as the number of classes increases.
