## Description

MTEB is the main benchmark suite used in [[behnamghader-2024-llmvec-2404-05961]] to evaluate sequence-level embedding quality across `56` datasets and `7` task categories.

## Key Contributions

- Serves as the paper's primary benchmark for unsupervised and supervised text embedding evaluation.
- Provides the leaderboard context for the paper's `56.80` unsupervised Mistral-7B result and `65.01` supervised Meta-Llama-3-8B result.
