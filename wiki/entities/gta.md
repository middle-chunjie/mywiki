## Description

GTA is a closed-set benchmark for evaluating general tool agents that is used in [[yang-2026-tooltree-2603-12740]]. It provides a fixed set of `14` typed APIs and supports both step-by-step and end-to-end evaluation.

## Key Contributions

- Serves as one of the main closed-set evaluations for ToolTree.
- Measures tool selection, argument prediction, planning, and execution quality.
