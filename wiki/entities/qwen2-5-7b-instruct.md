## Description

Qwen2.5-7B-Instruct is the main backbone model used for Tool-Light's strongest reported results. The paper evaluates both direct inference and multiple tool-integrated reasoning baselines on top of this model family.

## Key Contributions

- Serves as the backbone for the main Tool-Light system that reaches `58.0` average performance.
- Supports the paper's comparisons between prompting, Tool-Star, and Tool-Light training stages.
