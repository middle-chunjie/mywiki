## Description

The Abstraction and Reasoning Corpus is the visual reasoning benchmark used in [[tang-2024-code-2405-17503]] to test whether REx can guide LLM program synthesis on image-transformation puzzles.

## Key Contributions

- Provides a `40`-problem subset with human-written hypotheses for the paper's visual reasoning evaluation.
- Shows that REx can outperform the fixed-width Hypothesis Search baseline while reducing LLM calls.
