## Description

Boris Ginsburg is a researcher at [[nvidia]] and a co-author of [[ficek-2025-scoring-2502-13820]], which studies synthetic verification for code and reasoning models.

## Key Contributions

- Co-developed the benchmark transformation methodology used to derive ranked solution sets from HumanEval and MBPP.
- Co-authored the empirical analysis showing the advantage of reasoning models for synthetic test generation.
