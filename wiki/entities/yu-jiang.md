## Description

Yu Jiang is a researcher at Tsinghua University and a co-author of this paper. He participates in the study of how code data affects reasoning acquisition in large language models.

## Key Contributions

- Co-authored the stage-by-stage analysis of code data in pre-training and instruction tuning.
- Contributed to the interpretation of reasoning gains under CoT prompting and code-mixed training.
