## Description

SASRec is a self-attention-based sequential recommendation model introduced as a strong baseline and used as the architectural starting point for HPM in [[huang-2023-dual]].

## Key Contributions

- Provides the base transformer-style sequential encoder that HPM extends into a dual-transformer design.
- Serves as a central baseline showing the benefit of adding category-aware and relation-aware modeling.
