## Description

POJ-104 is the public C/C++ program dataset used in [[gui-2022-crosslanguage-2201-07420]] for single-language binary-source code matching. The paper describes it as containing about `50,000` programs and uses it to test robustness beyond the smaller cross-language dataset.

## Key Contributions

- Serves as the main benchmark for single-language `C++ binary -> C++ source` matching.
- Supports experiments under different compilers, architectures, and optimization levels.
- Shows XLIR reaching `0.85` F1 in the reported setup.
