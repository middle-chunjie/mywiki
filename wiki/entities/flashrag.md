## Description

FlashRAG is the retrieval toolkit whose Wikipedia retrieval settings are reused for Tool-Light's knowledge-intensive experiments. In this paper it functions as infrastructure for the search-based evaluation setup rather than as the core methodological contribution.

## Key Contributions

- Provides the retrieval configuration used with `E5-base-v2` on knowledge-intensive datasets.
- Anchors the paper's search-tool implementation details for the QA benchmarks.
