## Description

Fairseq is the sequence-modeling toolkit used to run the experiments in [[ahmad-2021-unified]]. The paper states that all experiments were implemented with this library.

## Key Contributions

- Provided the training and fine-tuning infrastructure for PLBART.
- Anchors the implementation side of the reported pretraining and downstream experiments.
