## Description

Alvin Cheung is a co-author of [[gong-2024-evaluation-2403-04814]] and works on evaluation methodology for code generation and code-editing models.

## Key Contributions

- Co-authors the SAFIM benchmark and its analysis of pretraining effects in code LLMs.
- Frames syntax-aware, repository-style evaluation as a better proxy for real development than standalone function synthesis.
