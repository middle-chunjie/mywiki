## Description

Ming Zhu is a researcher in the Department of Computer Science at [[virginia-tech]] (Arlington, VA). He is the first author of the CoST dataset and MuST-PT model for multilingual program translation.

## Key Contributions

- Led the construction of the CoST multilingual code snippet translation dataset covering 7 programming languages.
- Proposed MuST-PT, combining denoising auto-encoding and multilingual snippet translation pre-training for program translation.
- Demonstrated that snippet-level multilingual training generalizes to improve multiple baseline models on program-level translation.
