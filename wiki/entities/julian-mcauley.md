## Description

Julian McAuley is a co-author of [[guo-2023-longcoder-2306-14893]] and is affiliated with [[university-of-california-san-diego]] in the paper. In this source he is part of the collaboration that studies long-range code completion with sparse pre-trained language models.

## Key Contributions

- Co-authored the LongCoder method and the LCC benchmark study.
- Helped position the work against short-context and sparse-attention code completion baselines.
