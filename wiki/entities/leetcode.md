LeetCode is an online coding platform used in [[huang-2024-effibench-2402-02037]] as the source of efficiency-critical problems, topic tags, and candidate high-quality reference solutions.

## Key Contributions

- Provides topic tags that enable the benchmark's `11` algorithm subsets.
- Hosts discussion-forum solutions that are curated into executable [[canonical-solution]] baselines.
