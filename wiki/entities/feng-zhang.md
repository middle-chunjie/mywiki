## Description

Feng Zhang is a co-author of WildToolBench and is affiliated with Tencent HY according to the paper.

## Key Contributions

- Co-authored a benchmark showing that no evaluated model exceeds `14.45%` session accuracy on realistic tool-use tasks.
- Helped frame the benchmark around user behavior rather than purely synthetic task complexity.
