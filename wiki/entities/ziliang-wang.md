## Description

Ziliang Wang is a first author of StepSearch and is affiliated with [[sensetime]] in this paper.

## Key Contributions

- Co-authored the StepSearch framework that adds step-wise rewards to PPO for search-capable LLM agents.
- Helped build the MuSiQue-derived process-supervision dataset with sub-question search trajectories.
