## Description

QwQ-32B-Preview is the 32B large reasoning model used as the Search-o1 backbone. In this paper it provides the o1-like long reasoning behavior that the retrieval and refinement framework augments.

## Key Contributions

- Serves as the base model for all Search-o1 experiments.
- Supplies the long chain-of-thought behavior that agentic retrieval is designed to support.
- Provides the direct-reasoning and retrieval-augmented baselines that Search-o1 improves upon.
