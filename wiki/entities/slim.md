## Description

Slim is the single-agent long-horizon search framework introduced in [[yen-2025-lost-2510-18939]]. It combines lightweight search snippets, targeted browsing, and periodic trajectory summarization to control context growth.

## Key Contributions

- Separates search and browse into two explicit tools instead of bundling retrieval and full-page scraping.
- Uses periodic trajectory summarization to support longer searches with lower context and tool cost.
- Achieves stronger open-source performance on BrowseComp and HLE with fewer tool calls.
