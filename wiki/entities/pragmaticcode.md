## Description

PRAGMATICCODE is the repository-level Java dataset curated in this paper, consisting of buildable open-source repositories with development environments and dependencies preserved for evaluation.

## Key Contributions

- Contains `100` Java repositories that successfully initialize the required static-analysis tooling.
- Serves as the source corpus from which DOTPROMPTS is extracted for method-level completion.
