## Description

DyNet is the neural network toolkit used to implement the attentional correspondence model in [[yin-2018-mine-1805-08949]].

## Key Contributions

- Supports the encoder-decoder correspondence model that estimates `P(I | S)` and `P(S | I)`.
- Enables experiments with embedding size `256`, hidden size `512`, and dropout-based regularization.
