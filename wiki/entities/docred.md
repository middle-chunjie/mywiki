## Description

DocRED is a large-scale human-annotated document-level relation extraction dataset built from Wikipedia and Wikidata. In [[sun-2024-consistency-2401-13598]], it is one of the two main benchmarks for zero-shot document-level relation and triplet extraction.

## Key Contributions

- Provides the unseen-relation evaluation benchmark used for both ZeroDocRE and ZeroDocRTE.
- Supports comparisons across `m = 5` and `m = 10` unseen relation settings in the paper.
