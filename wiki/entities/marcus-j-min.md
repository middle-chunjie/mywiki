## Description

Marcus J. Min is a Columbia University researcher and co-author of [[ding-2024-semcoder-2406-01006]]. The paper places him on work spanning executable-code data curation and semantic-aware training for code models.

## Key Contributions

- Co-developed the PYX/PyX-R data pipeline used to teach semantic reasoning and self-refinement.
- Contributed to the evaluation of SEMCODER on code generation, execution reasoning, and debugging benchmarks.
