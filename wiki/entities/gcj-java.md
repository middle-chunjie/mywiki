## Description

GCJ-Java is a Java authorship-attribution dataset created by the authors from Google Code Jam submissions between 2015 and 2017.

## Key Contributions

- Contains `2,396` Java files from `74` authors with an average of `139` lines per file.
- Extends evaluation of style-based authorship attacks to a larger Java competition setting.
