## Description

Checkmate-in-One is a symbolic reasoning task drawn from the BIG-Bench suite in which the model must identify a move that immediately mates the opponent. The paper uses it to test whether meta-prompting improves exact-answer reasoning without task-specific exemplars.

## Key Contributions

- Provides one of the strongest text-only wins for meta-prompting, reaching `57.2` versus `36.4` for standard prompting.
- Shows that centralized expert orchestration can help even without Python execution.
