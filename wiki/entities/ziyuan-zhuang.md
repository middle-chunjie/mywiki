## Description

Ziyuan Zhuang is a researcher at the State Key Laboratory for Novel Software Technology, Nanjing University. Co-first author (equal contribution) of EfficientRAG.

## Key Contributions

- Co-proposed EfficientRAG, an iterative retriever that replaces LLM-based query generation with lightweight token-level classifiers for multi-hop question answering.
- Led the design and implementation of the Labeler+Tagger and Filter components.
