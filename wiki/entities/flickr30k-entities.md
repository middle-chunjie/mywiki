## Description

Flickr30K Entities is a text-layout dataset used in [[cho-2023-visual-2305-15328]] to train the layout-aware Vicuna model for VPGen. It is also used to name the `VPGen (F30)` training condition in the experiments.

## Key Contributions

- Supplies object-grounding supervision for text-to-layout learning.
- Defines the smaller training setup compared with the combined `F30+C+P` configuration.
