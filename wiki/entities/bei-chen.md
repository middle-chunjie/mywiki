## Description

Bei Chen is a researcher at Microsoft Corporation and a co-author of RepoCoder. She has contributed to multiple code intelligence papers at Microsoft, including CodeT and RepoCoder.

## Key Contributions

- Co-authored RepoCoder, the iterative retrieval-generation framework for repository-level code completion.
- Co-designed the RepoEval benchmark.
