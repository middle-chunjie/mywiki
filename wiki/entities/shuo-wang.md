## Description

Shuo Wang is a researcher at Tsinghua University and a co-author of [[li-2024-building]]. He contributes to work on retrieval-augmented code modeling and code understanding.

## Key Contributions

- Co-authored CONAN, a unified framework that couples dense retrieval with multi-document generation for coding assistance.
- Helped demonstrate that denoised retrieved code knowledge can improve both standard code models and code LLMs.
