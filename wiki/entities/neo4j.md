## Description

Neo4j is the graph database backend used by `Mem0^g` in [[chhikara-nd-mem]] to store entities, relations, and associated graph-memory metadata.

## Key Contributions

- Provides the structured storage layer for the graph-memory variant.
- Supports relation-centric retrieval over entity-relation triples.
- Enables persistence of timestamped relational memories beyond the immediate context window.
