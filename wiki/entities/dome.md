## Description

DOME is the supervised baseline system used for comparison in [[geng-2024-large]]. It is described as an intent-driven comment generator that uses attention to focus on code regions relevant to a requested intent.

## Key Contributions

- Provides the main state-of-the-art baseline that the paper compares against across all five comment intent categories.
- Helps quantify how much few-shot prompting and retrieval can replace supervised training on this task.
