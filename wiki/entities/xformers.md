## Description

xFormers is the transformer library used in [[gloeckle-2024-better-2404-19737]] to implement heterogeneous-batch self-speculative decoding experiments.

## Key Contributions

- Provides the implementation substrate for the paper's decoding-speed measurements.
- Enables evaluation of how extra MTP heads affect exact inference throughput in practice.
