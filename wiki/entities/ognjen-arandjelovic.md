## Description

Ognjen Arandjelović is a researcher affiliated in this paper with the University of St Andrews. He co-authors the agentic framework proposed for autonomous tool creation from scientific repositories.

## Key Contributions

- Co-introduces [[toolmaker]] as a reusable architecture for repository-grounded tool generation.
- Contributes to the paper's framing of tool creation as a systems problem involving environment state and execution feedback.
