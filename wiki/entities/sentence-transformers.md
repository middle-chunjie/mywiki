## Description

Sentence-Transformers is the embedding library used to source and fine-tune several baseline models in the BLANCA evaluation.

## Key Contributions

- Provides the encoder implementations used for several baseline sentence and paragraph models.
- Supplies evaluators and loss setups used in ranking, link prediction, and regression-style fine-tuning.
