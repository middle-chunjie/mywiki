## Description

HalluRAG is a question-answering hallucination detection dataset for RAG systems used as the second main benchmark in [[yeh-2026-lumina-2509-21875]].

## Key Contributions

- Provides answerable closed-domain QA instances for comparing Lumina against uncertainty, verbalization, and utilization baselines.
- Highlights Lumina's largest reported gains, including AUROC above `0.9` across all evaluated models on this dataset.
