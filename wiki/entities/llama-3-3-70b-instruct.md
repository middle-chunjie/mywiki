## Description

Llama-3.3-70B-Instruct is a 70-billion-parameter instruction-tuned language model from Meta (Llama 3.3 series). It is evaluated as a critic model on [[processbench]], achieving the highest average F1 among non-reasoning open-source models.

## Key Contributions

- Achieves F1 = 58.0 average on [[processbench]] via majority voting, outperforming Llama-3.1-70B-Instruct (52.7) and all non-QwQ open-source models outside the Qwen2.5-72B class.
- Strong GSM8K performance (82.9 F1) indicating solid error identification on simpler problems.
