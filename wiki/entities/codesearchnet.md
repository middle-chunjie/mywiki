## Description

CodeSearchNet is the large-scale code corpus used in [[huang-2021-cosqa-2105-13239]] as the source of Python functions with documentation. The paper also uses the CodeSearchNet Python corpus as a baseline training source for CodeBERT.

## Key Contributions

- Supplies the raw code pool from which CoSQA selects candidate answer functions.
- Serves as the baseline training data reference in the model comparison tables.
