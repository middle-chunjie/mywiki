## Description

Bayesmark is a benchmark framework for comparing Bayesian optimization methods on machine-learning hyperparameter tuning tasks. This paper uses Bayesmark as the main public benchmark for end-to-end evaluation of LLAMBO.

## Key Contributions

- Supplies `25` public HPO tasks formed from dataset-model pairs in the paper's main benchmark.
- Provides the search spaces used for models such as RandomForest, SVM, DecisionTree, MLP, and AdaBoost.
