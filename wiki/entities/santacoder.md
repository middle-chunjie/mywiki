## Description

SantaCoder is the 1.1B-parameter code language model used as the main open model baseline in this paper and the strongest base model in several comparisons.

## Key Contributions

- With MGD, its `score@6` compilation rate rises from `59.97` to `73.03`.
- Combined with RLPG or FIM, it demonstrates that prompt-side context and output-side monitoring are complementary.
