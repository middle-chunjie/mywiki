## Description

Chinchilla is the `70B`-parameter DeepMind language model introduced in [[hoffmann-2022-training-2203-15556]] to validate compute-optimal scaling by training on `1.4T` tokens.

## Key Contributions

- Demonstrates that a `4x` smaller dense model can beat Gopher when trained on substantially more data with the same compute budget.
- Provides the paper's core empirical evidence on MMLU, BIG-bench, reading comprehension, and closed-book QA.
