## Description

Yong Yu is a researcher at [[shanghai-jiao-tong-university]] and a co-author of [[long-2022-multiview]]. Yu is part of the senior author team behind the paper's investigation into semantic code representation for algorithm detection.

## Key Contributions

- Co-authored the MVG method for representing code with multiple semantic graph views.
- Contributed to the creation and evaluation of the [[alg-109]] benchmark.
- Supported the paper's analysis of semantic versus syntax-heavy representations.
