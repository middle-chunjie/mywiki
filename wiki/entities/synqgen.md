## Description

SynQGen is the public software toolkit released alongside [[almeida-2024-exploring]] for generating synthetic question-document datasets for information retrieval. It operationalizes the paper's document selection, question generation, and filtering pipeline.

## Key Contributions

- Packages the authors' synthetic-data generation workflow into an off-the-shelf tool.
- Supports application of the method to new unlabeled document collections.
