## Description

TREC is the question-type classification benchmark used in [[wang-2023-label]] for evaluating anchor-based ICL behavior and error diagnosis. The paper uses it especially for confusion analysis between label anchors.

## Key Contributions

- Serves as one of the four datasets for layerwise information-flow and intervention experiments.
- Provides the confusion-matrix case study linking anchor distances to real misclassification patterns.
