## Description

GPT-3.5-Turbo-0301 is one of the target black-box LLMs evaluated in [[jiang-2023-llmlingua-2310-05736]] for reasoning, dialogue, and summarization after prompt compression.

## Key Contributions

- Serves as the main closed-model evaluation target for LLMLingua in the reported GSM8K, BBH, ShareGPT, and Arxiv-March23 experiments.
- Demonstrates that `5x-20x` prompt compression can preserve strong task performance without model-side access.
