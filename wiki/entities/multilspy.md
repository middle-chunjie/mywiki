## Description

multilspy is the extensible Python library released with the paper for interfacing language models with language servers across multiple programming languages.

## Key Contributions

- Wraps language-server interactions behind a language-agnostic client abstraction.
- Supports the paper's implementation of monitors over Java and its generalization claims to C#, Rust, and Python tooling.
