## Description

Code Llama is the open-access code language model family evaluated in the paper for program optimization. The authors adapt `7B`, `13B`, and `34B` variants with prompting, fine-tuning, and performance-conditioned training.

## Key Contributions

- Serves as the main open model family benchmarked on PIE.
- Shows strong gains from retrieval-based prompting and performance-conditioned fine-tuning.
- The `13B` model reaches `5.65x` average speedup at `Best@8` with performance conditioning.
