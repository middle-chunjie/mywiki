## Description

Yu Zhou is a researcher at Nanjing University of Aeronautics and Astronautics (NUAA), China, working on software engineering, deep learning for code, and adversarial robustness of neural code models.

## Key Contributions

- Proposed ACCENT, an identifier-substitution adversarial attack framework for code comment generation models, demonstrating that DNN-based summarizers are fragile to small identifier renamings.
- Co-developed the masked training defense method that jointly optimizes original and identifier-masked objectives to reduce reliance on non-robust features.
