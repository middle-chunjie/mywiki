## Description

CodeExp is the dataset suite introduced in [[cui-2022-codeexp-2211-15395]] for explanatory code document generation. It contains raw, refined, and human-annotated partitions of Python function-docstring pairs.

## Key Contributions

- Provides `2,285,387` raw GitHub pairs, `158,024` refined pairs, and `13,186` human-annotated pairs.
- Supplies both training data and a high-quality evaluation resource for code explanation generation.
- Grounds the paper's analysis of how data filtering and data quality affect fine-tuned model performance.
