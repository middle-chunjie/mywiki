## Description

Pyro is the probabilistic programming library used in [[rodriguez-2021-evaluation]] to implement the Bayesian IRT models.

## Key Contributions

- Supports variational inference for latent skill and item parameters in DAD.
- Provides the probabilistic modeling substrate paired with PyTorch for the paper's experiments.
