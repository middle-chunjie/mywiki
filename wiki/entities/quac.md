## Description

QuAC is the conversational question answering benchmark used by [[li-2023-sm-2312-16511]] for both development experiments and hidden-test leaderboard evaluation.

## Key Contributions

- Serves as the target benchmark on which S2M is evaluated in unsupervised and supervised settings.
- Contains `13K` dialogs and `98K` questions according to the paper's dataset statistics.
- Provides the leaderboard on which DEBERTA-S2M reached first place at submission time.
