## Description

COFIC is the cleaned Java code search dataset released by [[sun-2022-importance-2202-06649]], built from GitHub code paired with filtered comments.

## Key Contributions

- Retains `1,048,519` comment-code pairs from `2,475,692` raw pairs after NLQF processing.
- Outperforms CodeSearchNet-based training in follow-up experiments with both DeepCS and CARLCS.
