## Description

Aider is a repository-level coding agent adapted in this paper from program repair to test generation. Its workflow includes repository indexing, context summarization, and validation before applying edits.

## Key Contributions

- Provides a code-agent baseline for the paper's comparison on SWT-Bench.
- Achieves `S = 12.7%` and `Delta C_all = 27.8%` in the reported experiments.
