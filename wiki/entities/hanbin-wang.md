## Description

Hanbin Wang is a researcher at Northeastern University and a co-author of [[li-2024-building]]. He is part of the team that develops CONAN for retrieval-augmented code assistance.

## Key Contributions

- Co-authored CONAN, which combines a structure-aware retriever with a Fusion-in-Decoder generator for code-related tasks.
- Helped validate the system across code generation, code summarization, code completion, and code retrieval benchmarks.
