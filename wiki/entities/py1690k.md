## Description

PY1690K is the large deduplicated Python corpus collected for [[izadi-2022-codefill-2202-06689]] from non-forked GitHub repositories with more than `20` stars. It contains about `1.7M` files and `425M` lines of code and is used exclusively for pre-training.

## Key Contributions

- Supplies large-scale unlabeled code for the multi-task pre-training stage.
- Expands the training distribution beyond the benchmark dataset and supports better generalization than training only on PY117K.
