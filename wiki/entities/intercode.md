## Description

InterCode is the benchmark and framework introduced by [[yang-2023-intercode-2306-14898]] for standardizing interactive coding with execution feedback across executable environments.

## Key Contributions

- Defines an extensible agent-environment API for interactive coding tasks.
- Packages Bash, SQL, and Python environments with execution-based rewards and reproducible Docker setups.
- Serves as a benchmark for studying prompting strategies, error recovery, and context discovery in code-generation agents.
