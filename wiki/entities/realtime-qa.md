## Description

REALTIME QA is the dynamic benchmark and online platform introduced by [[kasai-2024-realtime-2207-13332]] for evaluating question answering systems on current-world information. It releases weekly news-based questions and tracks baseline and participant submissions over time.

## Key Contributions

- Recasts QA evaluation around live weekly updates rather than static frozen datasets.
- Provides both benchmark data and a public evaluation interface with real-time baselines.
- Exposes how retrieval freshness affects downstream QA accuracy.
