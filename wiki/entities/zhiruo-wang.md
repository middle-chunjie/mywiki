## Description

Zhiruo Wang is a coauthor of [[wang-2023-executionbased-2212-10481]] affiliated with the Language Technologies Institute at [[carnegie-mellon-university]]. In this paper, Wang contributes to the construction and analysis of the ODEX benchmark for open-domain code generation.

## Key Contributions

- Coauthored ODEX, a benchmark with `945` NL-code pairs and `1,707` human-written tests.
- Helped frame open-domain code generation as an execution-based evaluation problem rather than a purely lexical matching task.
