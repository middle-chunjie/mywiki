## Description

choices13k is a large-scale risky-choice dataset used in the paper's decisions-from-description experiments, containing more than `13,000` choice problems and over one million human choices.

## Key Contributions

- Provides the main benchmark for evaluating whether LLM-derived representations can model decisions from description.
- Supplies enough behavioral scale for comparing CENTaUR against BEAST and raw LLaMA with cross-validated negative log-likelihood.
