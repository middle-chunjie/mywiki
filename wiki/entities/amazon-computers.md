## Description

Amazon-Computers is a co-purchase graph benchmark used in [[gong-2022-magcl-2212-07035]] to test unsupervised [[node-classification]].

## Key Contributions

- Serves as one of the non-citation datasets in the six-benchmark evaluation suite.
- MA-GCL reaches `88.83 +/- 0.3`, outperforming the strongest compared baseline reported in the table.
