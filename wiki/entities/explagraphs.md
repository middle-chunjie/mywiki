## Description

ExplaGraphs is a commonsense reasoning dataset reused in [[he-2024-gretriever-2402-07630]] as one component of [[graphqa]]. In this benchmark it is converted from triplets into a standard textual-graph format for graph question answering.

## Key Contributions

- Supplies small explanation graphs for commonsense reasoning, with average graph size `5.17` nodes and `4.25` edges.
- Shows that when graphs are already compact, G-Retriever can simply retain the full graph by setting retrieval `` `k = 0` ``.
