## Description

Daya Guo is a co-author of [[guo-2023-longcoder-2306-14893]] and is affiliated with [[sun-yat-sen-university]] in the paper. The work credits Daya Guo with the LongCoder method and the LCC benchmark for long-context code completion.

## Key Contributions

- Co-authored the sparse-attention design combining window, bridge, and global attention.
- Helped introduce the LCC benchmark for long-context code completion.
