## Description

TUDataset is a graph benchmark collection used by the paper Multi-Scale Subgraph Contrastive Learning for both the semantic-similarity study and downstream graph classification evaluation.

## Key Contributions

- Supplies the real-world graph datasets on which MSSGCL is evaluated.
- Provides both molecular and social-network benchmarks for testing whether multi-scale contrastive learning generalizes across graph types.
