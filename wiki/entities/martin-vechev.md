## Description

Martin Vechev is a researcher at ETH Zurich and senior co-author of the SWT-Bench paper. In this source he is associated with the benchmark's framing of test generation as a rigorous evaluation problem for code agents.

## Key Contributions

- Co-supervised the benchmark design linking issue text, bug-fix patches, and golden tests.
- Co-authored the experimental comparison between code agents and direct LLM baselines.
