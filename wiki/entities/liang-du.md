## Description

Liang Du is a Microsoft-affiliated researcher and co-author of [[lin-2023-batchprompt-2309-00384]], which studies how to reduce prompt overhead by batching multiple inputs per LLM call.

## Key Contributions

- Co-authored the self-weighted majority voting setup using model-generated confidence labels.
- Helped frame SEAS as an early-stopping mechanism over active batched samples.
