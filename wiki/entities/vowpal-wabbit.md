## Description

Vowpal Wabbit is the linear learning toolkit used in [[rodriguez-2021-evaluation]] to implement the logistic-regression baseline against the IRT models.

## Key Contributions

- Trains the feature-based linear baseline used to test whether IRT's benefits come only from simple feature engineering.
- Enables ablations over item IDs, subject IDs, text, statistics, and IRT-derived features.
