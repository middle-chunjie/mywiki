## Description

Jina AI is the industry affiliation for four of the five authors of [[kryvosheieva-2025-efficient-2508-21290]] and the organization behind the `jina-code-embeddings` model suite described in the paper. In the author block it appears as Jina AI GmbH in Berlin.

## Key Contributions

- Provides the institutional setting for the paper's code embedding research and model release.
- Supports the development of compact embedding models specialized for code retrieval workloads.
