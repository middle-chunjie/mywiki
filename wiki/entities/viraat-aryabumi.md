## Description

Viraat Aryabumi is a researcher affiliated with Cohere For AI and a corresponding author of [[unknown-nd-code]]. In this paper, he studies how code data affects general LLM pretraining performance beyond code generation.

## Key Contributions

- Co-authors the controlled ablation study over code proportion, quality, initialization, scale, and cooldown.
- Helps establish that code improves both reasoning and open-ended generation, not only coding benchmarks.
