## Description

Fast Downward is the planning system that provides the implementation of the LAMA baseline used in [[orseau-2018-singleagent-1811-10928]].

## Key Contributions

- Supplies the domain-independent planning infrastructure behind the paper's LAMA comparison.
- Provides the execution environment for the lazy GBFS baseline with preferred operators and queue alternation.
