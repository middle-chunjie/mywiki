## Description

BigBird is a sparse-attention Transformer model used as a token-classification baseline in [[li-2024-chulo-2410-11119]]. In this paper it is compared with ChuLo on long-document named entity recognition.

## Key Contributions

- Provides a long-input baseline for NER on CoNLL-2012 and GUM.
- Highlights how sparse-attention models degrade as document length grows on CoNLL.
