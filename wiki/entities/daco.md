## Description

DACO is the dataset and benchmark introduced in [[wu-2024-daco-2403-02528]] for application-driven data analysis over relational databases. It pairs stakeholder-oriented queries with multi-point findings and suggestions grounded in executable code traces.

## Key Contributions

- Provides `440` databases and `1,942` query-answer pairs, plus a `100`-example human-refined test subset.
- Supplies both weakly supervised GPT-4 traces and a benchmark for evaluating helpfulness-oriented analytical generation.
- Serves as the training and evaluation substrate for the paper's SFT and Daco-RL models.
