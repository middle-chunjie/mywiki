## Description

BLANCA is a benchmark suite for evaluating language models on textual artifacts about code, including forum discussions and class documentation.

## Key Contributions

- Defines five tasks spanning ranking, matching, and regression over code-related text.
- Serves both as an evaluation benchmark and as supervision for model fine-tuning.
- Exposes transfer patterns between forum-text tasks and code-property tasks.
