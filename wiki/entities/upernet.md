## Description

UperNet is the semantic segmentation framework used in [[yu-2024-mambaout-2405-07992]] to test MambaOut on ADE20K after ImageNet pretraining.

## Key Contributions

- Provides the evaluation head for the paper's ADE20K semantic segmentation experiments.
- Supplies the single-scale and multi-scale mIoU measurements used to assess dense prediction transfer.
