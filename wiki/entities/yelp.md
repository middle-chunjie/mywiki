## Description

Yelp is a public user-business recommendation dataset with category and review text used in [[ren-2024-representation]] for evaluation and pretraining transfer experiments.

## Key Contributions

- Provides a text-rich benchmark for testing whether LLM-generated profiles improve recommender representations.
- Supports the paper's downstream pretraining experiment using 2012-2017 data for pretraining and 2018-2019 data for transfer.
