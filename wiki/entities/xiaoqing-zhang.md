## Description

Xiaoqing Zhang is a researcher at Nanjing University of Aeronautics and Astronautics (NUAA), China, and co-author of the ACCENT adversarial robustness work on code comment generation.

## Key Contributions

- Co-developed the ACCENT adversarial example generation pipeline for code comment generation, including identifier extraction and candidate selection via word2vec.
