## Description

Wikidata5M is a Wikidata-derived knowledge graph subset used in [[yu-2024-kola-2306-09296]] as the basis for KoLA's known-data memorization setup and its linkage to the 2019 Wikipedia dump.

## Key Contributions

- Provides the factual triples used to reconstruct KoLA's known-data probing tasks.
- Supports frequency-based entity sampling for high-frequency and low-frequency memorization evaluation.
