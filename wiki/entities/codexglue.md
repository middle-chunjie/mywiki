## Description

CodeXGLUE is the public code benchmark suite used in [[guo-2023-longcoder-2306-14893]] to test whether LongCoder still performs well when the available code context is relatively short. The paper evaluates the PY150 and JavaCorpus completion tasks from this benchmark.

## Key Contributions

- Provides a short-context comparison point complementary to the new LCC benchmark.
- Shows that LongCoder remains state of the art even when long-range context is less necessary.
