## Description

DD is a graph classification dataset from TUDataset used by the paper Multi-Scale Subgraph Contrastive Learning for semantic-similarity analysis and downstream evaluation.

## Key Contributions

- Serves as one of the four datasets used to analyze multi-scale subgraph semantics.
- MSSGCL reaches `79.73 +- 0.44` in the unsupervised setting and `78.89 +- 0.18` with `10%` labels in the semi-supervised setting.
