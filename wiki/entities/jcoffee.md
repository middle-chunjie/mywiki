## Description

JCoffee is a helper tool used in [[sun-2022-code-2202-08029]] to make partial Java snippets compilable by adding missing class or method definitions before disassembly.

## Key Contributions

- Lets TranCS process incomplete code snippets that would otherwise fail compilation.
- Enables the instruction-generation stage to cover more real-world snippets.
