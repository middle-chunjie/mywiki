## Description

SkillsBench is a benchmark for measuring how much curated agent skills improve LLM-based agents on diverse terminal tasks. It pairs tasks with skill packages, oracle solutions, and deterministic verifiers so skill efficacy can be measured directly.

## Key Contributions

- Provides paired no-skills, curated-skills, and self-generated-skills evaluation conditions.
- Covers `84` evaluated tasks across `11` domains with deterministic verification and detailed failure analysis.
