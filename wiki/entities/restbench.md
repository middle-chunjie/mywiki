## Description

RestBench is an open-set REST API benchmark used in [[yang-2026-tooltree-2603-12740]] to evaluate tool planning over real-world endpoints in the TMDB and Spotify domains.

## Key Contributions

- Measures pass rate and win rate for open-set multi-step API planning.
- Tests whether ToolTree's pruning and grounded feedback remain effective beyond small fixed tool sets.
