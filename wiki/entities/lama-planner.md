## Description

LAMA is the heuristic planning baseline compared against LevinTS and sampling-based methods in [[orseau-2018-singleagent-1811-10928]].

## Key Contributions

- Solves all `1,000` Sokoban levels in the comparison table with `3,151,325` total expansions.
- Serves as the main domain-independent planner baseline for judging search effort and solution length.
