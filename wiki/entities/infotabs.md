## Description

InfoTabs is the tabular reasoning benchmark used in [[he-2022-rethinking-2301-00303]] to measure whether RR can combine table-derived background knowledge with external lexical knowledge from WordNet and ConceptNet.

## Key Contributions

- Provides `1,200` evaluation hypotheses from the development set after keeping only entailed and contradictory cases.
- Lets the paper test whether evidence-grounded reranking still helps when the premise is a structured table rather than free text.
