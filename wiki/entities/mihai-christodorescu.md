## Description

Mihai Christodorescu is a Google-affiliated researcher and co-author of [[hooda-2024-do-2402-05980]], contributing to the design of counterfactual mutations for diagnosing code-model behavior.

## Key Contributions

- Co-authored the mutation design for evaluating predicate-specific failures in code completion.
- Helped position CACP as a hard-label black-box framework rather than a white-box probing method.
