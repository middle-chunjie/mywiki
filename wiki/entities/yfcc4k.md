## Description

YFCC4k is a benchmark dataset of 4,000 geotagged images drawn from the Yahoo Flickr Creative Commons collection, used alongside Im2GPS3k to evaluate worldwide image geolocation methods (introduced by Vo et al., ICCV 2017).

## Key Contributions

- Serves as one of two primary evaluation datasets in the Img2Loc paper; Img2Loc (GPT-4V) achieves 14.11/29.57/41.40/59.27/76.88 % across the five distance thresholds, surpassing GeoGuessNet (prior SOTA).
- Provides a complementary test set with distinct Flickr image characteristics compared to Im2GPS3k.
