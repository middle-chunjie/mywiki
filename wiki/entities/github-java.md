## Description

GitHub-Java is a Java authorship-attribution dataset used in this paper to test robustness on real-world repositories.

## Key Contributions

- Contains `2,827` Java files from `40` authors with an average of `76` lines per file.
- Demonstrates that baseline models remain vulnerable to targeted imitation and untargeted hiding even outside competition data.
