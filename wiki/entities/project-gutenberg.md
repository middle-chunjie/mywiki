## Description

Project Gutenberg is the public-domain book repository from which the GutenQA source books are drawn. In this paper it serves as the upstream corpus providing long-form narrative documents for chunking and retrieval evaluation.

## Key Contributions

- Provides the 100 public-domain books used to build [[gutenqa]].
- Supplies narrative text whose structure motivates content-aware [[document-segmentation]].
- Grounds the benchmark in legally reusable material suitable for public release.
