## Description

New Bing (later rebranded Bing Chat / Bing Copilot) is Microsoft's LLM-augmented search engine, which integrates a GPT-4-class language model with live web search to generate concise, cited answers rather than returning a list of links.

## Key Contributions

- First large-scale commercial deployment of retrieval-augmented generation in a general web search engine.
- Demonstrated that the reader module — synthesizing retrieved documents into a fluent, cited response — could replace the traditional ranked list as the primary search result format.
- Set the product-level benchmark that subsequent LLM4IR research aimed to improve upon or analyze.
