## Description

Deze Wang is the first author of [[wang-2022-bridging-2112-02268]] and is affiliated with [[national-university-of-defense-technology]]. He studies how pretrained models can be adapted more effectively for source code understanding.

## Key Contributions

- Co-proposed the paper's data-centric adaptation recipe combining semantic-preserving augmentation, curriculum learning, and test-time augmentation.
- Helped validate the approach across algorithm classification, clone detection, and code search benchmarks.
