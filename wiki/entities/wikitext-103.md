## Description

WIKIText-103 is the in-domain language-modeling benchmark used to evaluate RETOMATON when the datastore is built from the same corpus as the base LM. The paper reports `103M/250K/250K` train/validation/test tokens.

## Key Contributions

- Serves as the main in-domain testbed for measuring perplexity against Fraction of Saved Searches.
- Provides the `103M`-entry datastore used for the paper's largest reported automaton.
