## Description

SHADE-Arena is an agentic harm benchmark of benign main tasks paired with harmful side tasks, used in [[kale-2026-reliable-2508-19461]] as one of the two main evaluation datasets. It covers tool-calling agents and provides the baseline environment for much of the paper's monitoring analysis.

## Key Contributions

- Supplies `17` main-side task pairs for monitor red teaming experiments.
- Serves as the main dataset for the paper's weak-to-strong monitoring study.
