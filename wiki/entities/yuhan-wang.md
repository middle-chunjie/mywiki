## Description

Yuhan Wang is a researcher at [[shandong-university]] and a co-author of [[shi-2025-retrieval-2503-01763]], which introduces ToolRet and ToolRet-train for large-scale tool retrieval evaluation.

## Key Contributions

- Co-develops the benchmark construction pipeline that standardizes heterogeneous tool-use datasets into retrieval tasks.
- Contributes to the analysis connecting retriever quality with downstream pass rate in tool-using LLM agents.
