## Description

Codex is OpenAI's code generation model family, used in [[bavarian-2022-efficient-2207-14255]] as an architectural and dataset reference point for the paper's code-domain pretraining setup.

## Key Contributions

- Anchors the claim that a single FIM-trained model can subsume specialized completion-only coding assistants.
