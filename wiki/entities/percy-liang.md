## Description

Percy Liang is a researcher at [[stanford-university]] and a co-author of the DSP paper. In this work, he helps articulate natural-language programming as a practical systems interface for retrieval-augmented in-context learning.

## Key Contributions

- Co-authors DSP as a modular framework for deliberate, task-aware LM-RM control flow.
- Helps position demonstration bootstrapping and explicit search planning as alternatives to rigid prompting pipelines.
