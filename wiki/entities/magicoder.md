## Description

Magicoder is the family of open-source code language models introduced in [[wei-2023-magicoder-2312-02120]]. The paper defines Magicoder as OSS-INSTRUCT-tuned base models and MagicoderS as their second-stage Evol-Instruct variants.

## Key Contributions

- Shows that `7B`-scale open models can approach or beat much larger baselines on code benchmarks.
- Demonstrates that OSS-INSTRUCT and Evol-Instruct are complementary rather than redundant.
- Provides open weights, data, and code for follow-up research.
