## Description

Gemini CLI is Google's open-source terminal agent harness evaluated in SkillsBench. In the benchmark it achieves the highest with-skills pass rate when paired with Gemini 3 Flash.

## Key Contributions

- Provides the best with-skills result in the paper at `48.7%` pass rate with Gemini 3 Flash.
- Uses an explicit skill-activation interface, so the self-generated-skills condition is not evaluated for this harness.
