## Description

srcML is the code-transformation tool used in this paper to perform semantics-preserving rewrites on individual C, C++, and Java source files.

## Key Contributions

- Enables the implementation of coding-style imitation and hiding transformations.
- Supports multi-language program rewriting needed for the paper's four datasets.
