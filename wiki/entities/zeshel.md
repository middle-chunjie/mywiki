## Description

ZeShEL is a zero-shot entity-linking benchmark used in [[unknown-nd-adaptive-2405-03651]] to evaluate cross-encoder `k`-NN search on held-out domains. In this paper, YuGiOh and Star Trek provide the small- to medium-scale entity inventories for testing recall and downstream accuracy.

## Key Contributions

- Supplies the entity-linking evaluation setting for the paper's cross-encoder search experiments.
- Provides the YuGiOh (`10,031` items) and Star Trek (`34,430` items) domains used in the main analysis.
