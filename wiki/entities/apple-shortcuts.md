## Description

Apple Shortcuts is the workflow-automation environment used as the main real-world data source in [[unknown-nd-workflowllm]]. It exposes built-in and third-party actions that can be assembled into executable workflows.

## Key Contributions

- Supplies the underlying workflow substrate from which WorkflowBench seed data is collected.
- Illustrates the paper's target difficulty, with real workflows averaging `70.4` actions and nested control flow.
