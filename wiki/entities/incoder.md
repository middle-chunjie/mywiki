## Description

InCoder is a code infilling model family used in [[bavarian-2022-efficient-2207-14255]] as a closely related prior baseline and benchmark reference.

## Key Contributions

- Provides the single-line and multi-line infilling benchmark setup that this paper extends.
- Serves as a comparison point for large-scale decoder-only infilling performance.
