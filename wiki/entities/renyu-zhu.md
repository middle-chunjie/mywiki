## Description

Renyu Zhu is a graduate student at the School of Data Science and Engineering, [[east-china-normal-university]], Shanghai. He is the lead author of PGNN-EK, a program understanding model combining graph neural networks with external knowledge.

## Key Contributions

- First author of PGNN-EK, proposing the S-AST representation and partitioning-based GNN for code understanding.
- Co-released the BCB-F dataset, a harder benchmark for code clone detection split by functionality.
