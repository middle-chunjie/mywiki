## Description

Bottleneck-Minimal-Indexing is the public code repository linked by [[du-2024-bottleneckminimal-2405-10974]] for reproducing the paper's generative retrieval experiments and indexing procedures.

## Key Contributions

- Provides an implementation of BMI and its comparisons against HKmI, HRI, and LSHI.
- Makes the paper's query-aware indexing pipeline and evaluation setup inspectable and reusable.
