## Description

Graph4Code is a machine-interpretable knowledge graph linking Python code, documentation, and developer discussions; BLANCA is built on top of this resource.

## Key Contributions

- Supplies linked artifacts from `1.3M` Python programs for benchmark construction.
- Provides the class relations and usage signals needed for the hierarchy and usage tasks.
