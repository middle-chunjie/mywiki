## Description

ClueWeb09-B is the web document collection used for the TREC Web Track ad hoc ranking experiments in [[qiao-2019-understanding-1904-07531]]. In this paper it provides the document corpus over which neural and feature-based rerankers are compared.

## Key Contributions

- Supplies the document collection for the paper's ad hoc retrieval benchmark.
- Is used with top-`100` candidate documents retrieved by Galago SDM and evaluated under `10`-fold cross-validation.
