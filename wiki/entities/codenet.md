## Description

CodeNet is a large-scale dataset of programming problems and submissions. In [[dinh-2023-large-2306-03438]], it provides problem statements that are paired with FixEval submission data to build buggy-FixEval prompts.

## Key Contributions

- Supplies problem descriptions for constructing realistic [[buggy-code-completion]] instances.
- Helps align program-repair traces with executable completion prompts in the benchmark pipeline.
