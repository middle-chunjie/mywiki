## Description

NeuLab is Graham Neubig's research lab at the Language Technologies Institute, Carnegie Mellon University, focusing on neural approaches to natural language processing, multilingual NLP, and code intelligence.

## Key Contributions

- Developed and released CodeBERTScore along with five language-specific CodeBERT models (Java, Python, C, C++, JavaScript) on the Hugging Face Hub; models have been downloaded more than 1,000,000 times.
- Produced prominent work on NL→Code evaluation, multilingual code generation, and DocPrompting.
