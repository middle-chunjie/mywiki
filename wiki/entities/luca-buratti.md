## Description

Luca Buratti is an IBM Research author on [[ding-2022-dissimilarity-2110-03868]]. In this paper he contributes to DISCO's pretraining design for code similarity, vulnerability detection, and clone detection.

## Key Contributions

- Co-authored the DISCO framework and its evaluation against pretrained code-model baselines.
- Helped formulate a data-efficient pretraining strategy built around program contrasts.
