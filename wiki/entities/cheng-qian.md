## Description

Cheng Qian is a researcher at [[university-of-illinois-urbana-champaign]] and the first author of [[qian-2025-smart-2502-11435]]. In this paper he helps define tool overuse and build the SMART/SMART-ER framework for boundary-aware agent reasoning.

## Key Contributions

- Co-introduced the SMART paradigm for balancing parametric reasoning and tool use.
- Co-developed the SMART-ER dataset and its step-level justification pipeline.
- Helped analyze the efficiency and calibration effects of strategic tool use in agents.
