## Description

Devign is the defect-detection benchmark used in [[unknown-nd-code-2309-09980]] to test whether execution-aware pre-training transfers to vulnerability-oriented classification. The paper reports accuracy on this dataset.

## Key Contributions

- Serves as the main defect-detection evaluation set for FuzzPretrain.
- Shows that dynamic execution signals help on a classification task beyond retrieval.
