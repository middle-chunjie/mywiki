## Description

PYX is the executable Python training corpus introduced in [[ding-2024-semcoder-2406-01006]] for teaching code models approximate, abstract, and operational program semantics.

## Key Contributions

- Provides `32k` executable synthetic samples filtered for parsability and successful execution.
- Supplies the semantic supervision used to train SEMCODER for code generation and execution reasoning.
