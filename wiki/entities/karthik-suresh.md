## Description

Karthik Suresh is a researcher in the Department of Computer Science at [[virginia-tech]] (Arlington, VA) and co-author of the CoST multilingual code translation dataset and MuST-PT model.

## Key Contributions

- Co-developed the CoST parallel code snippet dataset with snippet-level alignment across 7 programming languages.
- Co-authored the MuST-PT multilingual program translation model.
