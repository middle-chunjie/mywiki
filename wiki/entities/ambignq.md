## Description

AmbigNQ is the ambiguous open-domain QA benchmark used in [[ma-nd-query-2305-14283]] to test whether query rewriting better matches underspecified information needs. The paper evaluates the first `1000` test examples.

## Key Contributions

- Serves as a main benchmark where trainable rewriting reaches `47.80` EM / `60.71` F1.
- Supports retrieval analysis showing hit ratio improves from `76.4%` to `82.2%` with trainable rewriting.
