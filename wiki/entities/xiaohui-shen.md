## Description

Xiaohui Shen is a ByteDance-affiliated author of [[jin-2024-mvadapter-2301-07868]]. Shen is part of the team proposing MV-Adapter for parameter-efficient video-text retrieval.

## Key Contributions

- Co-authored the adapter architecture that augments frozen CLIP encoders.
- Helped demonstrate efficiency gains in storage and GPU memory usage.
