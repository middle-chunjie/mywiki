## Description

DIV2K is the high-quality image database used in [[guo-2024-refir-2410-05601]] for nearest-neighbor reference retrieval in the wild. Its images provide the external visual knowledge that ReFIR injects into restoration.

## Key Contributions

- Supplies the retrieval corpus for the paper's RealPhoto60 experiments.
- Enables testing whether semantically retrieved high-quality images can improve restoration without retraining the base LRM.
