## Description

Zhensu Sun is the first author of [[sun-2022-importance-2202-06649]] and was affiliated with [[monash-university]] at the time of publication. He works on data quality and neural code search.

## Key Contributions

- Co-proposed the two-stage filtering framework combining syntactic rules and VAE-based semantic filtering.
- Led the empirical study showing that noisy comment queries materially hurt neural code search.
