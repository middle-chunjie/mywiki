## Description

Baoquan Zhong is a researcher affiliated with ByteDance Seed and a co-author of ReTool. The paper places Zhong on a project about reinforcement learning for code-interpreter-augmented LLM reasoning.

## Key Contributions

- Co-authored the data-construction and RL framework used for strategic tool use.
- Contributed to the behavioral analysis of code pass rate, invocation timing, and self-correction.
