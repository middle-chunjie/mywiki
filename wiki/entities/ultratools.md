## Description

UltraTools is an out-of-domain tool-use and tool-retrieval dataset used in [[xu-2024-enhancing-2406-17465]] as one component of TR-bench.

## Key Contributions

- Provides `1,000` evaluation instructions over a tool set of `498` tools in the paper's OOD setting.
- Helps test whether retrieval improvements transfer beyond the ToolBench training distribution.
