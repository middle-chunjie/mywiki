## Description

Lakshya A Agrawal is a researcher at Microsoft Research Bangalore and the lead author of this paper on monitor-guided decoding for repository-level code generation.

## Key Contributions

- Co-introduced monitor-guided decoding as an interface between code LMs and static analysis.
- Helped curate the PRAGMATICCODE dataset and DOTPROMPTS benchmark used for evaluation.
