## Description

ARCADE is a multi-turn data-science programming benchmark in notebook context, used in [[allamanis-2024-unsupervised-2402-08699]] to test whether [[round-trip-correctness]] correlates with standard benchmark scores beyond HumanEval.

## Key Contributions

- Serves as a second narrow-domain calibration benchmark for SYNTHESISRTC.
- Demonstrates that RTC also tracks performance in notebook-style contextual coding tasks.
