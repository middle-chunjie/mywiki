## Description

Typilus is a graph-based neural model for Python type hint prediction and serves as one of the main deep-learning baselines in the HiTYPER paper.

## Key Contributions

- Provides neural type recommendations that HiTYPER can consume in Top-k mode.
- Serves as a strong baseline for overall, rare-type, and return-value inference.
