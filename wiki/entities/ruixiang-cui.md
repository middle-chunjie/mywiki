## Description

Ruixiang Cui is a researcher at Microsoft and a co-first-author of AGIEval (equal contribution with Wanjun Zhong). The AGIEval GitHub repository is hosted under the `ruixiangcui` handle. Cui contributed to benchmark design, data collection, and evaluation.

## Key Contributions

- Co-first-authored AGIEval, contributing equally to benchmark construction and model evaluation.
- Maintained the public GitHub repository for AGIEval data and model outputs.
