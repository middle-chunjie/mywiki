## Description

Code Alpaca is the `20K` instruction-following code dataset used as the seed corpus in [[luo-2023-wizardcoder-2306-08568]] before iterative evolution.

## Key Contributions

- Serves as the starting point for WizardCoder's code-domain [[evol-instruct]] pipeline.
- Provides the simple instruction-response pairs later expanded into `38k` to `98k` evolved datasets.
