## Description

Somesh Jha is a University of Wisconsin-Madison and Google-affiliated researcher listed on [[hooda-2024-do-2402-05980]], where he co-authors the CACP evaluation framework for code LLMs.

## Key Contributions

- Co-developed the overall framework linking Programming Concept Predicates to measurable mutation effects.
- Helped position the work as a diagnostic benchmark for concept understanding rather than only end-task coding accuracy.
