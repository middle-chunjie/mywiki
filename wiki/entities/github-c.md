## Description

GitHub-C is a C-language authorship-attribution dataset constructed by the authors from GitHub repositories contributed between November and December 2020.

## Key Contributions

- Contains `2,072` C files from `67` authors with an average of `88` lines per file.
- Supports evaluation of both adversarial-example attacks and coding-style attacks on real-world C code.
