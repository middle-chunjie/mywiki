## Description

Codex CLI is OpenAI's terminal coding agent harness and one of the evaluated systems in SkillsBench. The paper reports competitive raw performance but weaker practical use of provided skills than the other harnesses.

## Key Contributions

- Serves as the harness for GPT-5.2 in the benchmark.
- Reaches `44.7%` with curated skills but often acknowledges skills without fully exploiting them.
