## Description

BrowseComp-en is a difficult English benchmark for web-based information seeking and one of the main evaluation targets in [[li-2025-websailor-2507-02592]].

## Key Contributions

- Serves as the paper's main stress test for high-uncertainty web reasoning.
- Highlights a large gap between direct inference and browsing-based agents.
- Shows the clearest open-source improvement for WebSailor, which reaches `12.0` at `72B`.
