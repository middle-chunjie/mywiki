## Description

Humanity's Last Exam is the expert-level benchmark used in [[yen-2025-lost-2510-18939]] to test whether long-horizon search helps with difficult reasoning-heavy questions. The paper evaluates on the text-only subset.

## Key Contributions

- Provides a complementary evaluation setting where web search must support domain reasoning rather than only exhaustive lookup.
- Shows that Slim improves over open-source baselines on reasoning-heavy long-horizon tasks as well as on BrowseComp.
