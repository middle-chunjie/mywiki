## Description

TranCS is the code-search system introduced in [[sun-2022-code-2202-08029]]. It retrieves code by translating Java bytecode into natural-language-like descriptions and embedding those translations in the same space as user queries.

## Key Contributions

- Introduces context-aware instruction translation for code retrieval.
- Uses shared word mapping to reduce embedding mismatch between code and natural language.
- Outperforms DeepCS and MMAN on Java CodeSearchNet benchmarks.
