## Description

Hinrich Schütze is a co-author of [[yan-2026-memoryr-2508-19828]] affiliated with Ludwig Maximilian University of Munich and the Munich Center for Machine Learning. In this paper, he is part of the collaboration on reinforcement-learned memory agents.

## Key Contributions

- Co-authored Memory-R1, which learns both memory editing and memory-conditioned answer generation.
- Contributed to the study's evaluation of adaptive memory behavior in LLM agents.
