## Description

CodeParrot is the large-scale code dataset used for the paper's decoder-only next-token-prediction experiments. In [[he-2023-simplifying-2311-01906]], it serves as the main environment for measuring training-speed differences between Pre-LN, SAS, and SAS-P Transformer blocks.

## Key Contributions

- Provides the causal language-modeling benchmark for the paper's 18-layer GPT-style experiments.
- Supports the paper's runtime, longer-training, and depth-scaling comparisons for simplified Transformer blocks.
