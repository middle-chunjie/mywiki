## Description

Python Programming Puzzles is a benchmark of code-generation puzzles with programmatic correctness constraints. In this paper it is used to evaluate whether meta-prompting plus tool use improves functionally correct code synthesis.

## Key Contributions

- Shows that meta-prompting with Python reaches `45.8`, exceeding standard prompting (`31.1`) and zero-shot CoT (`36.3`).
- Tests whether decomposition and expert-style prompting transfer beyond pure text reasoning into executable program synthesis.
