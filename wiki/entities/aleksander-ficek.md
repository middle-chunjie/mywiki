## Description

Aleksander Ficek is a researcher at [[nvidia]] and the first author of [[ficek-2025-scoring-2502-13820]], which introduces scoring benchmarks for synthetic verification in code and reasoning.

## Key Contributions

- Co-developed the benchmark-construction recipe that converts coding datasets with oracle tests into ranked scoring datasets.
- Co-authored the empirical comparison between generated-test verifiers, reward models, and reasoning models.
