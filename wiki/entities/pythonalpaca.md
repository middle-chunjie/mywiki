## Description

PythonAlpaca is a large Python question-answer dataset used as the code-centric corpus in [[unknown-nd-contextaugmented-2601-20810]]. The authors preprocess it into function-level artifacts to build the code-side Programming Knowledge Graph.

## Key Contributions

- Supplies `143,000` programming question-answer pairs for code-centric retrieval experiments.
- Yields `115,000` extracted Python functions that become nodes in the PKG.
