## Description

UCLID5 is the formal verification language targeted by this paper's SPEAC pipeline. It is used for modeling transition systems and checking whether those models satisfy formal specifications.

## Key Contributions

- Serves as the target VLPL in the paper's main case study.
- Defines the compilation target for the child-language programs produced by SPEAC.
- Provides the benchmark domain in which parse rate and semantic correctness are measured.
