## Description

Mask R-CNN is the detection and instance segmentation framework used in [[yu-2024-mambaout-2405-07992]] to evaluate MambaOut as a transferable backbone on COCO.

## Key Contributions

- Provides the backbone transfer setting for the paper's COCO experiments.
- Produces the box and mask AP metrics used to compare MambaOut against VMamba and related baselines.
