## Description

Yiqiao Jin is the lead author of [[jin-2023-code]] and is affiliated with [[georgia-institute-of-technology]]. The paper positions Jin as a contributor to graph-based recommendation methods for open-source development.

## Key Contributions

- Co-introduced the code recommendation task for open-source developers.
- Co-developed CODER, which fuses code semantics, project hierarchy, and multi-behavior user signals.
