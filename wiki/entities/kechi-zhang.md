## Description

Kechi Zhang is a researcher at the Key Lab of High Confidence Software Technology, MoE, [[peking-university]], working on code representation learning and program analysis. He is the lead author of HiT (Hierarchy Transformer).

## Key Contributions

- Proposed the hierarchical embedding framework decomposing CST paths into global and local hierarchy components.
- Designed the Hierarchy Transformer (HiT) integrating CST-derived structural embeddings into sequence-based code models.
- Conducted empirical evaluation across 8 datasets on code classification, clone detection, and method name prediction.
