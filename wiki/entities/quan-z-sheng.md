## Description

Quan Z. Sheng (also known as Michael Sheng) is a professor at Macquarie University, Sydney, Australia, specializing in web services, data management, and machine learning applications in NLP.

## Key Contributions

- Co-authored the first survey on adversarial attacks on NLP deep learning models.
- Contributed framing of the threat model taxonomy and defense analysis.
