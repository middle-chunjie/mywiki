## Description

BERT-base-uncased is the encoder backbone used as the dependency discriminator in [[unknown-nd-tool-2508-05152]]. It classifies ordered tool pairs into forward dependency, no dependency, or reverse dependency.

## Key Contributions

- Provides the base encoder for the paper's three-class discriminator.
- Supplies pairwise dependency predictions that define the tool graph used by TGR.
