## Description

Jiaqi Liu is a co-author of [[xia-2026-skillrl-2602-08234]] and is affiliated with [[unc-chapel-hill]] in the paper. He is part of the collaboration building a skill-centric alternative to raw memory for agent RL.

## Key Contributions

- Helps evaluate the framework on interactive agent benchmarks and search-augmented QA tasks.
