## Description

M&M's is a multimodal, multi-step tool-use benchmark used in [[yang-2026-tooltree-2603-12740]] for closed-set planning evaluation. It provides `33` APIs spanning vision, text, and arithmetic tasks.

## Key Contributions

- Tests ToolTree under longer-horizon multimodal tool composition.
- Reports both step-by-step and multi-step execution metrics.
