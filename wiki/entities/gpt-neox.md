## Description

GPT-NeoX is the open-source training library used to train the Pythia models. In this paper it provides the autoregressive Transformer implementation, parallelism stack, and checkpointing infrastructure.

## Key Contributions

- Trained all Pythia models used in the suite.
- Supported data and tensor parallelism for multi-scale experiments.
- Provided the implementation base for reproducible release of training artifacts.
