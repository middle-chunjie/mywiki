## Description

CoNLL-2003 is the labeled benchmark used in [[behnamghader-2024-llmvec-2404-05961]] for chunking, named-entity recognition, and part-of-speech tagging probes over frozen model representations.

## Key Contributions

- Provides the word-level evaluation used to test whether LLM2Vec improves contextualized token representations.
- Supplies training, validation, and test splits of roughly `14,000`, `3,250`, and `3,450` samples for the paper's linear probing setup.
