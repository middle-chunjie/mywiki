## Description

Ashish Hooda is a researcher affiliated with the University of Wisconsin-Madison in [[hooda-2024-do-2402-05980]], where he co-authors the CACP framework for evaluating programming-concept understanding in code LLMs.

## Key Contributions

- Co-developed the CACP framework for black-box evaluation of code models.
- Helped build the counterfactual benchmark derived from HumanEval, MBPP, and CodeContests.
