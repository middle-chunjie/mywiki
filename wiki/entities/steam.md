## Description

Steam is a public game recommendation dataset with user textual feedback used in [[ren-2024-representation]] to test RLMRec on a denser interaction setting without explicit rating filters.

## Key Contributions

- Supplies one of the paper's three core evaluation datasets for ranking quality and efficiency.
- Illustrates how noisy or irregular review text can still be turned into useful semantic profiles for recommendation.
