## Description

ExecEval is the execution framework used in [[gong-2024-evaluation-2403-04814]] to run unit tests for algorithmic-block and control-flow code completions.

## Key Contributions

- Provides the execution environment for testing generated code rather than scoring only by text overlap.
- Supports the paper's emphasis on functional correctness for most of the benchmark.
