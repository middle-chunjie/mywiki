## Description

Jinjun Peng is a Columbia University researcher and co-author of [[ding-2024-semcoder-2406-01006]]. In this paper, Peng contributes to training code language models with explicit supervision over program semantics.

## Key Contributions

- Co-authored SEMCODER, a semantic-aware code LM built on DeepSeekCoder-6.7B.
- Helped frame monologue reasoning as a training signal for execution reasoning and debugging.
