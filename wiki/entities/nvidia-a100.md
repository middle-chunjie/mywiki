## Description

NVIDIA A100 is the GPU platform used to train the dialog retriever in [[levy-2023-chatting-2305-20062]], where the authors report four A100 nodes.

## Key Contributions

- Provides the compute environment for training the retriever in `114` seconds per epoch.
- Grounds the paper's implementation details for large-batch dialog retrieval training.
