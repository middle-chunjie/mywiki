## Description

Dejiao Zhang is a researcher at [[aws-ai-labs]] and the first author of [[unknown-nd-code-2402-01935]]. In this paper he leads the development and analysis of the CodeSage code representation learning framework.

## Key Contributions

- Co-designed the two-stage CodeSage pretraining recipe that combines identifier deobfuscation, full-mask MLM, and bimodal contrastive learning.
- Led the large-scale empirical study across zero-shot code search and code classification benchmarks.
- Analyzed why full masking, hard negatives, and hard positives matter for scaling encoder-based code embeddings.
