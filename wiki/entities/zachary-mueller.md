## Description

Zachary Mueller is a researcher at Hugging Face and a co-author of [[wei-2024-selfcodealign-2410-24198]]. In this paper he contributes to the self-alignment pipeline and empirical evaluation for transparent code LLM instruction tuning.

## Key Contributions

- Co-authored the SelfCodeAlign study on self-generated instruction data for code large language models.
- Helped evaluate the aligned model across code generation, class generation, data science, and code editing benchmarks.
