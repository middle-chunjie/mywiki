## Description

Shuvendu K. Lahiri is a researcher at Microsoft Research Redmond and a co-author of this paper on semantically guided code generation.

## Key Contributions

- Co-authored the formulation of monitors that use static analysis to constrain decoding.
- Contributed to the discussion of richer semantic properties such as typestates and session-type protocols.
