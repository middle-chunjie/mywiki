## Description

Michihiro Yasunaga is a researcher at Stanford University and a co-author of [[guti-rrez-2024-hipporag-2405-14831]].

## Key Contributions

- Contributed to the paper's neurobiologically inspired framing of long-term memory for LLM retrieval systems.
- Helped validate that graph-based retrieval improves downstream QA when paired with the same reader backbone.
