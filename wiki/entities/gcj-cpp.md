## Description

GCJ-C++ is a C++ authorship-attribution dataset derived from Google Code Jam and used in this paper to evaluate both baseline and RoPGen-enabled models.

## Key Contributions

- Contains `1,632` C++ files from `204` authors with `8` files per author.
- Provides one of the main evaluation settings for both adversarial-example and coding-style attacks.
