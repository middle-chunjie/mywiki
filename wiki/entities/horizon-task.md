## Description

The horizon task is a repeated choice paradigm used to study exploration versus exploitation, and in this paper it serves as the main decisions-from-experience benchmark.

## Key Contributions

- Tests whether the adapted model reproduces human exploratory behavior rather than merely matching aggregate reward.
- Supports both population-level evaluation and participant-level random-effects analysis.
