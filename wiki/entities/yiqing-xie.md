## Description

Yiqing Xie is a co-author of CodeRAG-Bench and is listed with Carnegie Mellon University in this paper. Xie contributes to the benchmark's task integration and retrieval-oriented evaluation setup for code generation.

## Key Contributions

- Co-authored CodeRAG-Bench, which unifies retrieval, generation, and execution-based evaluation across diverse coding tasks.
- Helped build and analyze the benchmark's multi-source retrieval corpus and canonical-document evaluation pipeline.
- Contributed to the large-scale comparison of retrievers and generators for retrieval-augmented code generation.
