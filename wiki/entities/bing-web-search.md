## Description

Bing Web Search is the search backend used in the paper's mathematical-reasoning evaluation setup. The system pairs it with a browser mechanism to prevent overly long search interactions.

## Key Contributions

- Supplies the external retrieval capability for math-oriented tool calls.
- Forms one of the two concrete tool types instantiated in the Tool-Light experiments.
