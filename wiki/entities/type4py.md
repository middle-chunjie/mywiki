## Description

Type4Py is a hierarchical neural network for Python type prediction and is used as a principal neural baseline in the HiTYPER evaluation.

## Key Contributions

- Supplies neural Top-k type candidates that HiTYPER can validate and refine.
- Provides a comparison point for argument, local-variable, return-value, and rare-type prediction.
