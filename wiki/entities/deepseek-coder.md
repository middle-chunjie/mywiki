## Description

DeepSeekCoder is the `6.7B` code language model used as the initialization checkpoint for SEMCODER in [[ding-2024-semcoder-2406-01006]]. It serves as the paper's most direct pretrained baseline.

## Key Contributions

- Provides the base checkpoint from which semantic-aware training starts.
- Serves as the reference model for measuring gains in execution reasoning and code generation.
