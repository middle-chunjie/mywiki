## Description

Claude 2 is an API-accessed large language model evaluated in [[cheng-2024-blackbox-2311-04155]]. The paper shows that BPO can improve Claude 2's pairwise win rate without any model retraining.

## Key Contributions

- Serves as a black-box evaluation target demonstrating that BPO works on closed models.
- Shows an average GPT-4-judged win-rate gain of `+8.8` when paired with BPO-optimized prompts.
