## Description

Xingwei Qu is a researcher affiliated with ByteDance Seed and a co-author of ReTool. In this work, Qu is part of the team developing a tool-integrated RL framework for LLM mathematical reasoning.

## Key Contributions

- Co-authored a rollout design that alternates natural-language reasoning, code generation, and interpreter feedback.
- Contributed to the analysis of emergent code self-correction and more efficient reasoning trajectories.
