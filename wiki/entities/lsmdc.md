## Description

LSMDC (Large Scale Movie Description Challenge) is a video-text benchmark containing 118,081 short clips from 202 movies, each annotated with a caption from the movie script or audio description; test set is 1000 clips from held-out movies.

## Key Contributions

- Benchmark for CrossCLR: Text→Video R@1 = 15.0, MdR = 18.0 (outperforms MMT which uses additional Audio, OCR, and Face modalities: R@1 = 13.2, MdR = 21.0).
