## Description

CodeLlama-Python is the `7B` code model used as the main base backbone in [[wei-2023-magicoder-2312-02120]]. Magicoder-CL and MagicoderS-CL are produced by instruction-tuning this model on OSS-INSTRUCT and then Evol-Instruct data.

## Key Contributions

- Serves as the starting checkpoint for the paper's main Magicoder results.
- Provides the baseline scores that quantify gains from open-source-inspired instruction tuning.
