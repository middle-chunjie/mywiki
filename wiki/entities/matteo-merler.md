## Description

Matteo Merler is a researcher at [[aalto-university]] and a co-author of [[dainese-2024-generating-2405-15383]] on LLM-guided synthesis of code world models.

## Key Contributions

- Co-authored the GIF-MCTS framework for iterative code generation with generate, improve, and fix actions.
- Contributed to the empirical evaluation of Code World Models on CWMB, APPS, and RTFM.
