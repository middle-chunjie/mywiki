## Description

DeepSeek-R1-Distill-Qwen-1.5B is a distilled reasoning model cited as both an upstream ancestor of [[deepscaler-1.5b-preview]] and a direct baseline in the paper's experiments. It represents the starting reasoning-model lineage from which the controllable L1 system is derived.

## Key Contributions

- Provides the distilled reasoning backbone underlying the DeepScaleR model family.
- Serves as a direct baseline in the paper's budgeted reasoning comparisons.
