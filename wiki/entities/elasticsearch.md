## Description

Elasticsearch is the search platform used in [[katsis-2025-mtrag-2501-03468]] to host the benchmark corpora and run ELSER-based passage retrieval during data creation and evaluation.

## Key Contributions

- Stores the four mtRAG corpora as passage indexes for retrieval experiments.
- Provides the retrieval backend used by the live annotation system.
