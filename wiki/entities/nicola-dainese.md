## Description

Nicola Dainese is a researcher at [[aalto-university]] and the lead author of [[dainese-2024-generating-2405-15383]], which introduces GIF-MCTS for synthesizing Code World Models.

## Key Contributions

- Co-authored the proposal of Code World Models as executable Python world models for planning.
- Co-authored GIF-MCTS, the generate-improve-fix search strategy built around MCTS and execution feedback.
