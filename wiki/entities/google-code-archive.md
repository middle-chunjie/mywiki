## Description

Google Code Archive is the repository collection used in [[shrivastava-2023-repositorylevel-2206-12839]] to build a Java code-completion dataset with reduced overlap risk relative to GitHub.

## Key Contributions

- Supplies the `47` Java repositories used for train, validation, and test splits.
- Supports the paper's attempt to reduce contamination from likely Codex training data.
