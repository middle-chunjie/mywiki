## Description

Lambda^2 is a synthesis tool discussed in [[balog-2017-deepcoder-1611-01989]] that combines enumerative search with deduction to prune program search spaces. DeepCoder uses it with a sort-and-add schedule over predicted DSL functions.

## Key Contributions

- Provides a deduction-enhanced synthesis baseline for evaluating neural guidance.
- Shows strong speedups when combined with DeepCoder's learned function ranking.
