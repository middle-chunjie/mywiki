## Description

Juanjuan Shen is a researcher at Nanjing University of Aeronautics and Astronautics (NUAA), China, and co-author of the ACCENT study on adversarial robustness for code comment generation.

## Key Contributions

- Co-authored work on adversarial attacks and masked training defense for neural code comment generation models.
