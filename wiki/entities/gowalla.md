## Description

Gowalla is a public location check-in recommendation dataset used in [[ren-2023-disentangled]] to evaluate collaborative filtering under sparse implicit feedback.

## Key Contributions

- Serves as one of the three main benchmarks for DCCF.
- Provides `50,821` users, `57,440` items, and `1,172,425` interactions with density `4.0e-4`.
