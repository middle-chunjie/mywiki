## Description

Amazon Tübingen is an Amazon research lab in Germany focusing on machine learning and computer vision; Yi Zhu and Peter Gehler are affiliated with it, and Mohammadreza Zolfaghari conducted the CrossCLR work there as an intern.

## Key Contributions

- Host institution for the CrossCLR project (cross-modal contrastive learning for video representations).
