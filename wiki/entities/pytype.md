## Description

Pytype is Google's Python static type inference and checking tool, used in the paper as a baseline for the static component of type inference.

## Key Contributions

- Serves as an industrial static baseline for argument and return type inference.
- Highlights the larger coverage achieved by HiTYPER's static inference component.
