## Description

Kaiyang Zhou is a researcher affiliated with Nanyang Technological University (NTU), Singapore, known for work on domain generalization, vision-language models, and prompt learning.

## Key Contributions

- Co-author of [[zhang-nd-what]], contributing to the prompt retrieval framework for visual in-context learning.
- Co-authored influential work on CoOp (prompt learning for vision-language models) and domain generalization surveys referenced in [[zhang-nd-what]].
