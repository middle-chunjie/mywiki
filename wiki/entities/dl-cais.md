## Description

DL-CAIS is one of the two baseline deep-learning-based authorship attribution methods evaluated in the paper. It uses lexical program features, recurrent neural networks, fully connected layers, and random forest prediction.

## Key Contributions

- Serves as a strong baseline and as a base model for RoPGen-enabled training.
- Achieves high clean accuracy on all four evaluated datasets.
- Exposes how token-oriented attribution behaves under style-based attacks and robust training.
