## Description

Game of 24 is an arithmetic reasoning task in which a system must combine four numbers into an expression that evaluates to `24`. In this paper it is one of the clearest demonstrations of the benefit of tool-enabled meta-prompting.

## Key Contributions

- Serves as a task where Python integration dramatically changes performance: meta-prompting improves from `11.0` without Python to `67.0` with Python.
- Highlights the value of decomposition plus external computation over direct prompting baselines.
