## Description

Sriram K. Rajamani is a researcher at Microsoft Research Bangalore and a co-author of the paper proposing monitor-guided decoding.

## Key Contributions

- Co-authored the overall framework for combining repository-scale static analysis with code LM decoding.
- Helped position the method as a practical alternative to purely prompt-based repository-context integration.
