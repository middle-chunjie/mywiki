## Description

Claude Code is Anthropic's terminal agent harness and one of the three commercial agent interfaces evaluated in SkillsBench. The paper highlights its native skill integration and strong gains when paired with curated skills.

## Key Contributions

- Serves as the harness for Claude Opus 4.5, Opus 4.6, Sonnet 4.5, and Haiku 4.5 in the benchmark.
- Shows some of the largest curated-skill gains, including `+23.3` points for Opus 4.5.
